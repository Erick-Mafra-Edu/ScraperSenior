                    Análise de desempenho
                 
                 
                O Workflow disponibiliza uma forma de analisar o desempenho dos seus processos, esta análise é efetuado com base em alguns dados das instâncias do processo e das tarefas. Para que seja possível realizar esta análise o Workflow disponibiliza alguns dados e possibilita que sejam definidos dados de cada processo também. 
                Veja como disponibilizar esses dados na Disponibilização de dados, e após a disponibilização ser realizada, é necessário então realizar a Extração dos dados para que eles sejam analisados.
