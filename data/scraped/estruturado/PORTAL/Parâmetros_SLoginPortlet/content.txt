            Parâmetros SLoginPortlet
            Alguns parâmetros estão disponíveis para utilização em configurações de links. São parâmetros relativos ao colaborador associado, ao usuário logado e ao cálculo ativo. 
            Parâmetros:
            
                Data 
				 de início do cálculo ativo do usuário logado = {inical}
                Data 
					 de fim do cálculo ativo do usuário logado = {fimcal}
                Código 
					 do tipo do cálculo ativo do usuário logado = {tipcal}
                Código 
					 da empresa do colaborador selecionado = {addemp}
                Código 
					 do tipo colaborador do colaborador selecionado = {addtip}
                Código 
					 de cadastro do colaborador selecionado = {addcad}
                Código da empresa do usuário 
					 logado = {numemp}
                Código do tipo colaborador 
					 do usuário logado = {tipcol}
                Código de cadastro do usuário 
					 logado = {numcad}
                Data de início da apuração 
					 do cálculo ativo do usuário logado = {iniapu}
                Data de fim da apuração do 
					 cálculo ativo do usuário logado = {fimapu}
            
            Qualquer 
					 um dos parâmetros acima poderão ser utilizados para 
					 preencher campos do formulário Web 5.0, basta incluir 
					 um "&" antes do "&adicionarParametros=true" 
					 e incluir um novo campo da Web 5.0 que receberá um dos 
					 parâmetros acima sempre no formato (&nome_campowebg5={parâmetroportal}).
            Comando de ativação
            Para 
					 que os parâmetros efetivamente sejam buscados, eles 
					 precisam de um comando de ativação. Sem esse comando 
					 os parâmetros não serão passados para o relatório/formulário. 
					 Cada parâmetro de entrada possui parâmetros de ativação 
					 distintos:
            
                Parâmetro 
					 de ativação (&adicionarParametros=true): 
					  Aciona os seguinte parâmetros: {addemp}, 
					  {addtip}, {addcad};
                Parâmetro 
					 de ativação (&buscarParametros=true):  Aciona 
					 os seguinte parâmetros: {inical}, {fimcal}, {tipcal}, 
					 {numemp}, {tipcol}, {numcad}, {iniapu}, {fimapu}.
