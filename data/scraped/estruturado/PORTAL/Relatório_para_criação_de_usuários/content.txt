            Relatório para criação de usuários
            O relatório para criação de usuários permite todos os colaboradores terem acesso ao Portal Corporativo.
O relatório procura colaboradores que não têm usuário associado e cria usuários para eles. Para os usuários que já possuem colaborador associado, o relatório exibe e identifica que são usuários já existentes, ou seja, não foram criados na execução do relatório.

            O relatório cria os grupos Portal Corporativo - Gestores e Portal Corporativo - Colaboradores, mas caso ele não seja executado, os grupos devem ser criados manualmente.
            

Para todas as operações o relatório considera as variáveis de entrada de abrangências definidas:
            
                EAbrEmp
                EAbrTcl
                EAbrCad
                EAbrLoc

            
            Sendo que a abrangência utilizada para a criação de usuários através deste relatório é do tipo 2 (Empresa/Tipo Colaborador/Cadastro). Para alterá-la na criação, é necessária a customização no relatório.
            Informar parâmetros de entrada do relatório
                
                    Modo de execução: define se o relatório cria ou simula a criação dos usuários.
                    Formato do nome dos usuários: defina o formato do nome dos usuários.
                    Formato da senha dos usuários: defina o formato da senha dos usuários.
                    Listar usuários: defina quais usuários serão listados.
                    Enviar dados dos usuários criados por e-mail: permite enviar os dados do usuário criado para o e-mail do seu respectivo colaborador. Para que o relatório envie o e-mail, o colaborador deve ter um e-mail comercial cadastrado.
                    Definir usuários gestores: defina se o relatório vai criar usuários e parametrizar com permissão de gestor. 
                    Gestores: defina quais colaboradores tem usuário criado como gestor.
                    Listar abrangência: defina se a abrangência dos usuários será listada no relatório.
                    Empresa: defina a abrangência de empresas fazem parte da criação de usuários.
                    Tipo Colaborador: defina a abrangência de tipo de colaborador a fazer parte da criação de usuários.
                    Colaborador: defina a abrangência de cadastros a fazer parte da criação de usuários.
                    Local: defina a abrangência de local a fazer parte da criação de usuários.
