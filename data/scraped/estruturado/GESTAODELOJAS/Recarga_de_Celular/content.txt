                Recarga de Celular
                Conceito     
                É um serviço oferecido na loja e tem a finalidade de vender recargas para telefones celulares diretamente no PDV, sem necessitar de um pedido ou de uma pré-venda.
                Utilização     
                O serviço é realizado através de integração com os sistemas integradores TEF - SiTEF e TEF V$Pague, conforme configurações do PDV.As operadoras de telefonia deverão estar configuradas nos integradores TEF para que a recarga possa ser efetuada e, posteriormente, devem ser cadastradas no Retaguarda em cadastro de operadoras de telefonia.
                Processos
                O processo para efetuar uma recarga é feito exclusivamente no terminal de caixa PDV.
                Botão Recarga
                Para realizar uma recarga, o operador de caixa deverá acionar o botão Recarga celular para que o sistema se conecte respectivo integrador. 
                Insira as informações na seguinte sequência com a utilização de Sitef:
                
                    Operadora de telefonia entre as opções disponíveis na tela;
                    Número do telefone com DDD (dois dígitos, ex.: 47);
                    Valor da recarga (será exibido o valor mínimo e máximo permitido por operadora);
                    Solicite confirmação do cliente no Pin Pad;
                     Selecione a forma de pagamento entre as opções disponíveis na tela.
                
                
                    Insira as informações na seguinte sequência com a utilização de V$Pague:
                
                    Operadora de telefonia entre as opções disponíveis na tela;
                    Número do telefone com DDD (dois dígitos, ex.: 47);
                    Valor da recarga (será exibido o valor mínimo e máximo permitido por operadora);
                    Solicite confirmação do cliente no Pin Pad.
                
                
                    Nota:
                    
                    O sistema de PDV poderá disponibilizar as formas de pagamento abaixo para recebimento da recarga, na utilização da  SiTEF:
                    
                        Dinheiro
                        Cheque
                        Cartão de débito
                        Cartão de crédito
                    
                    Para a V$Pague, poderá ser utilizada a opção:
                    
                        Dinheiro
                    
                    Essas formas de pagamento são disponibilizadas através de configurações no SiTEF, V$Pague e PDV.
                
                Cheque
                 Ao utilizar a forma de pagamento  cheque, deverão ser informados os seguintes dados: CMC7, CPF/CNPJ e nome do sacado. Caso o CPF/CNPJ informado não esteja cadastrado no sistema, o usuário poderá realizar o cadastro alimentando as informações solicitadas pelo sistema. 
                
                    Observação: 
                    O preenchimento dos dados solicitados pelo sistema não são obrigatórios para efetuar uma recarga. O usuário poderá teclar "Enter" para passar adiante.
                
                Após a mensagem de "transação aprovada", tecle "Enter" para que o ECF faça a impressão do comprovante.
                O sistema  emitirá  comprovante não fiscal para registrar a entrada de valores no caixa referente à recarga de celular juntamente com o relatório gerencial em duas vias:
                
                    O comprovante será emitido utilizando o totalizador configurado no ECF para recarga de celular;
                    Caso identificado o cliente, seus dados serão impressos no comprovante.

                    Nota
                    
                    Através do  parâmetro dinâmico (do Gestão Empresarial | ERP) TEF.DUPLICARECIBORECARGACELULAR é possível indicar se o texto do recibo de recarga de celular deve ser duplicado, permitindo assim a segunda via, pois no caso do VsPague somente uma via é gerada. Para isso, o valor do parâmetro deve ser "Sim".
                
                Cartão de crédito/dédito Para pagamento em cartão crédito ou débito, sempre deverá ser utilizada a transação via TEF, pois não é possível utilizar POS para vender recarga.
                
                    Nota
                    
                    Caso o parâmetro dinâmico RECARGA.MOTIVOLANCAMENTO esteja configurado no Gestão Empresarial | ERP para a filial, as vendas de recarga de celular gerarão como motivo enviado para a tesouraria o valor definido para o parâmetro dinâmico.
