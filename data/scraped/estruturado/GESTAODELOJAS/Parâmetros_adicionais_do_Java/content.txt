            Parâmetros adicionais do Java 
            Conceito
            Informar parâmetros para o Java de forma que o sistema Gestão de Lojas seja executado com customizações específicas do ambiente.
            Dentro de cada instalação do sistema, há arquivos com a extensão .vmoptions. Nesses arquivos devem ser configurados os seguintes parâmetros adicionais:
            
                
                
                
                    
                        Parâmetro
                        
                        Descrição
                        
                    
                    
                        -Djava.net.SocketTimeoutException=300000
                        Define o tempo em milisegundos de duração da conexão entre os sistemas que utilizam RMI. Esse parâmetro pode resolver problemas de queda de conexão por time-out entre os sistemas Retaguarda e PDV
                    
                    
                        -Xms1024m
                        Define a memória inicial que o sistema irá utilizar onde 1024 é = 1gb de memória ram.
                    
                    
                        -Xmx2048m
                        Define a memória limite que o sistema irá utilizar onde 2048 é = 2gb de memória ram.
                    
                
            
            
                Observações
                
                Estes parâmetros devem ser ajustados de acordo com o ambiente em que sistema está instalado levando em consideração a utilização de recursos de máquina por outros programas instalados e em execução.
                Após a configuração desses parâmetros o sistema deve ser iniciado.
                Esse parâmetros devem ser adicionados/alterados sob a supervisão do suporte Senior.
            
            Para mais informações acesse a documentação da Oracle.
