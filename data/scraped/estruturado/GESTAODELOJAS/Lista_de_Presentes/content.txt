            Lista de Presentes
            Conceito
            Criar e editar listas de presentes, sejam essas de casamento, aniversário etc.
            
                Nota
                
                A lista de presentes não contempla a adição de serviços de intermediação. Ex: Garantias estendidas, seguros furto/roubo/danos elétricos, cartão presente.
            
            Processo
            Consultar listas
            
                Acesse a tela de Lista de Presentes através do caminho Mercado > Gestão de vendas > Lista de presentes, no Retaguarda;
            
            
                Clique em Pesquisar. O sistema buscará no Gestão Empresarial | ERP todas as listas de presentes vigentes e ativas.
            
            
                Observação
                
                Caso deseja-se buscar por uma lista ou cliente específico, é possível informar no campo Código, o código da lista de presentes. Ou, ainda, consultar pelo cliente principal da lista.
                Caso seja retornada uma lista inativa e/ou fora do período de vigência:
                
                    o sistema permitirá editar a lista já criada; ou
                    o sistema permitirá criar uma nova lista.
                
            
            Cadastrar uma lista
            É possível realizar o cadastro de uma nova lista através do botão Incluir.
            As informações a seguir se aplicam à grade Dados Gerais.
            
                Preencha os seguintes campos (obrigatórios):
            
            
                Cliente Principal;
                Descrição;
                Data Evento.
            
            O sistema dispõe, ainda, dos seguintes campos (cujo preenchimento é opcional):
            
                Cliente Secundário;
                Código (esse campo será gerado pelo Gestão Empresarial | ERP. Esse campo tem como objetivo identificar se está editando uma lista já existente ou se está criando uma nova);
                Situação (Opções: Ativa, Inativa. Padrão: Ativa);
                Observações;
                Convidados;
                Local.
            
            
                Observação
                
                Para que a lista de presentes possa ser salva, deverá conter, pelo menos, um item incluso na grade Produtos Solicitados. Além de informar, obrigatoriamente, o Cliente Principal e a Descrição.
                A lista de presentes não é gravada na base de dados do Retaguarda. Ela é enviada diretamente ao Gestão Empresarial | ERP.
            
            Editar uma lista
            É possível alterar uma lista de presentes através do botão Alterar. Porém, não é possível alterar o cliente principal nem o código da lista. As demais informações, por sua vez, podem ser alteradas normalmente.
            Incluir um item na lista
            As informações a seguir se aplicam à grade Produtos Solicitados.
            
                Digite o produto desejado no campo Produto;
            
            
                Clique sobre o botão de consulta ou tecle Enter. O(s) produto(s) será(ão) listado(s) na tela;
            
            
                Selecione o produto;
            
            
                Clique no botão Incluir.
            
            
                Nota
                
                Antes de clicar no botão Incluir, selecione através do botão Aceita Similar (Opções: Ativa, Inativa. Padrão: Ativa), se aceita produtos similares ou não.
            
            
                Observação
                
                Ao tentar incluir, na lista de presentes, produto solicitado, o sistema buscará o preço padrão daquele determinado produto. Caso não esteja configurado, o produto não deve ser adicionado à lista.
            
            Remover um item da lista
            As informações a seguir se aplicam à grade Produtos Solicitados.
            
                Digite o produto desejado no campo Produto;
            
            
                Clique sobre o botão de consulta ou tecle Enter. O(s) produto(s) será(ão) listado(s) na tela;
            
            
                Selecione o produto;
            
            
                Clique no botão Remover.
            
            O botão Remover, por sua vez, possui dois comportamentos:
            
                caso o produto ainda não foi integrado ao Gestão Empresarial | ERP: exclusão física do produto;
                caso o produto tenha sido integrado ao Gestão Empresarial | ERP: exclusão lógica do produto. Ou seja, apenas é alterada a situação para Cancelado. Isso se faz necessário, pois caso a lista já esteja no Gestão Empresarial | ERP, é possível que em outra filial alguém esteja informando uma venda referente àquele produto solicitado.
            
            A situação dos produtos pode ser consultada pelo botão Exibir, através da opções:
            
                Produtos em aberto;
                Produtos cancelados;
                Produtos vendidos;
                Todos os produtos.
            
            
                Observação
                
                A lista sempre será apresentada na grade em ordem decrescente (menor para maior), referente ao valor do produto.
                Ao realizar a pesquisa de uma lista de presentes, a tela sempre apresentará a lista filtrando-a pelos itens na situação Produtos em aberto.
            
            Registrar na lista um item vendido
            As informações a seguir se aplicam à grade Produtos Vendidos.
            
                Preencha os campos Produto, Vendedor, Nota/Cupom, Convidado e Data/Hora;
            
            
                Clique em Incluir.
            
            
                Nota
                
                Para que as informações sejam apresentadas, torna-se obrigatório o preenchimento de todos os campos. Essas informações podem, também, ser consultadas na tela de Nota fiscal ou Cupom Fiscal, do Gestão Empresarial | ERP.
            
            Ao realizar a gravação dessas informações, o produto mudará sua situação, ficando como Vendido. Com isso, não será mais permitido realizar a venda desse produto pertencente à lista.
            Caso haja convidados que queiram presentar após a data do evento, a lista estará como Vencida. Porém, será permitido editá-la.
            Após o processo da loja retirar os presente da lista, ela poderá ser Inativada. Para inativar a lista, pode-se alterar a situação dela para Inativa através do campo Situação ou, ainda, através do botão Inativar, presente no rodapé da tela.
