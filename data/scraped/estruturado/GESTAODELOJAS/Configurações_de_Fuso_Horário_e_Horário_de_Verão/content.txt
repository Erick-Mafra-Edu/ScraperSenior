            Configurações de Fuso Horário e Horário de Verão
            Para o sistema Gestão de Lojas, existem verificações a serem efetuadas nas estações de trabalho e parametrizações a serem realizadas no sistema para adequação dos fusos horários e utilização do horário de verão, dependendo do documento fiscal utilizado nos PDVs.
            
                Importante
                
                Caso sejam utilizados outros periféricos diferentes do ECF, deve-se verificar junto ao fabricante do mesmo se eventualmente há alguma parametrização específica para horário de verão.
            
            Para todas as estações de trabalho e servidores
            Para todas as estações de trabalho onde são executados componentes do sistema Gestão de Lojas (Retaguarda e PDVs), deve-se definir o horário de acordo com o fuso horário utilizado pela Filial, levando-se em conta se há ou não a aplicação do horário de verão. A geração dos registros no sistema será baseada no horário das estações (bem como a geração de arquivos XML dos documentos eletrônicos).
            
                Observações
                
                
                    Após ajuste do horário no servidor, é necessário reiniciar o serviço "Integrador Retaguarda".
                    Para a Sefaz-RS, quando o horário de verão está vigente, o fuso horário passa a ser -02:00. Verificar, em caso de rejeições, se o fuso horário no XML está sendo enviado corretamente e ajustar conforme necessidade.
                
            
            Para os PDVs que utilizem solução PAF-ECF
            Além da parametrização do horário nas estações onde os PDVs estão instalados, é necessário, também, ativar o horário de verão dos ECFs, em PDVs presentes em localidades onde existe a incidência do horário de verão.
            Para o ajuste do horário de verão do ECF, é necessário acessar o aplicativo PDVConf que está localizado na pasta de instalação do PDV Senior.
            
                Atenção
                
                •	O ajuste para ativar o horário de verão só poderá ser feito após a emissão de uma redução Z. Portanto, deverá ser ajustado no final do dia após fechamento de caixa.
                •	O ajuste para desativar o horário de verão só poderá ser feito 1 (uma) hora após a emissão de uma redução Z.
            
            1.	Acesse o PDVConf;
            2.	Abra a guia Programação e utilize o botão Mudar Horário de Verão;
            3.	Confirme a entrada do horário de verão clicando em Sim.
            
                Importante
                
                A Senior não é responsável pela alteração de horários de ECFs que eventualmente estejam com horário incorreto. Este tipo de procedimento deverá ser verificado junto a assistência técnica autorizado de ECF.
            
            
                Para os PDVs que utilizem solução NFC-e e SAT-CFe
            
            Deve-se garantir que o horário das estações de trabalho esteja correto, pois estes horários são utilizados para geração das NFC-e/SAT-CFe.
            Além disso, no caso de utilização de equipamento SAT, deve-se verificar junto ao Fabricante do equipamento SAT utilizado se há algum procedimento específico a ser adotado para alteração do horário do equipamento.
            
                Para os Integradores Matriz e Filiais
            
            Os horários gerados nos arquivos de integração utilizados pelo sistema serão respeitados pelos Integradores. Neste caso, basta conferir a parametrização dos fusos horários corretamente nos servidores onde os componentes estão instalados, garantindo que os fusos utilizados condizem com os fusos da Matriz e de cada Filial.
