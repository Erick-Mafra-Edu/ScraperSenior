            Cadastro de cliente no PDV NFC-e / SAT 
            Conceito
            Permite  cadastrar novos clientes no PDV NFC-e / SAT.
            
                Observação
                
                Através do parâmetro dinâmico FISCAL.NFCE.HABILITAR.ENVIO.TAG.CONSUMIDOR, é possível definir se será enviado o CPF do cliente à Sefaz ao final da venda no PDV NFC-e. 
            
            Processo 
            Para cadastrar, deve-se clicar no botão F12 - Identifica Cliente a partir da tela de Recebimento no PDV e, em seguida no botão F3 - Cadastrar.
            
            
            
            
            Caso o usuário não tenha acesso para cadastrar o cliente, será solicitada autorização:
            Exemplo
                
                    
                    
                
            
            Logo após, serão solicitados os dados do cliente:
            Exemplo
                
                    
                    
                
            
            Ao lado esquerdo, os dados informados serão exibidos:
            Exemplo
                
                    
                    
                
            
            Ao informar o CEP, será sugerido o endereço (caso esteja cadastrado). Se o usuário não souber o CEP, poderá consultar por meio do botão F3 - Consulta CEP:
            Exemplo
                
                    
                    
                    Caso o CEP seja informado manualmente, é permitido informá-lo nos seguintes formatos, caso contrário será apresentada mensagem de "CEP Inválido": 
                    
                        96500000 – utilizando somente os 8 dígitos direto;
                        96.500-000 – utilizando (.) ponto e (-) hífen.
                    
                
            
            Serão solicitados os dados para pesquisa do CEP na tela:
            Exemplo
                
                    
                    
                
            
            Serão exibidos os CEPs encontrados e o usuário poderá refinar a pesquisa informando mais dados conforme solicitado na tela.
            Exemplo
                
                    
                    
                
            
            O usuário poderá navegar entre os registros usando as setas para cima e para baixo do teclado, para então, selecionar o CEP desejado.
            Exemplo
                
                    
                    
                
            
            É permitido navegar entre os dados alterados, usando as teclas seta para cima e seta para baixo.
            Para gravar o cadastro deve-se clicar no botão F8 - Gravar.
