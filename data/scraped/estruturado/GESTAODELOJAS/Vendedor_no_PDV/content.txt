            Vendedor no PDV
            Definindo atalho para identificação do vendedor
            É possível definir um atalho numérico para a identificação do vendedor durante a venda. Como padrão, o configurado é o número "5", que representa o atalho F5. Esse atalho é disponibilizado após a entrada no botão F12 - Identifica, da venda.
            Antes de configurá-lo, verifique a disponibilidade do atalho na tela, pois outras identificações podem estar utilizando-o.
            A configuração do atalho para identificação do vendedor pode ser realizada no Gestão Empresarial | ERP, na tela Configuração de parâmetros (F000PPD): "Tecla.BotaoIdentificaVendedor" = "5".
            Reconhecendo o vendedor da venda
            Ao iniciar uma nova venda, é possível visualizar o vendedor atual ao lado direito, acima do Status do PDV. Como padrão, o vendedor inicial é o próprio operador de caixa.
            
            
            Identificando o vendedor no início da venda
            Quando estiver configurado, será aberta, automaticamente, no início de cada venda, uma tela para ser selecionado o vendedor. Essa tela não será aberta caso seja importado um pedido.
            A configuração para identificar o vendedor no início da venda pode ser realizada no Gestão Empresarial | ERP, na tela Configuração de parâmetros (F000PPD): "Venda.IdentificaVendedorNoInicioVenda" = "1".
            
            
            Alterando o vendedor
            O vendedor pode ser alterado a qualquer momento durante a venda. A identificação do vendedor se dá pelo botão F12 - Identifica.
            
            
            O sistema se comporta de duas formas diferentes, dependendo da configuração de múltiplos vendedores estar ativa ou não.
            
                Caso esteja configurado para múltiplos vendedores:após a alteração do vendedor, apenas os próximos itens adicionados estarão vinculados a ele. Essa opção permite que seja definido um vendedor diferente por item.
                Caso não esteja configurado para múltiplos vendedores:após a alteração do vendedor, todos os itens estarão/serão vinculados a ele. Ou seja, se houver algum vendedor diferente no item anterior, este será substituído.
            
            A configuração para múltiplos vendedores pode ser realizada no Gestão Empresarial | ERP, na tela Configuração de parâmetros (F000PPD): "Venda.MultiplosVendedores" = "1".
            Visualizando vendedores por item
            Para visualizar qual vendedor está ligado a cada item, pode ser utilizado o atalho CTRL + F1. Só é possível utilizar essa função caso esteja configurado a venda para múltiplos vendedores.
            A configuração para múltiplos vendedores pode ser realizada no Gestão Empresarial | ERP, na tela Configuração de parâmetros (F000PPD): "Venda.MultiplosVendedores" = "1".
            
            
            Imprimindo o vendedor na venda
            Caso esteja configurado, é possível realizar a impressão do vendedor no documento de venda.
            
                Para PDV PAF, apenas o último vendedor selecionado é impresso nas observações do cupom fiscal;
                Para PDV NFC-e, todos os vendedores são impressos na observação do DANFE, separados por vírgula.
            
            A configuração para imprimir o vendedor no documento de venda pode ser realizada no Gestão Empresarial | ERP, na tela Configuração de parâmetros (F000PPD): "Venda.ImprimeVendedorCupom" = "1".
            Vendas com Pedido
            Não importando se a configuração de múltiplos vendedores estiver ativada ou não, o PDV respeita a quantidade de vendedores selecionados no Retaguarda durante a construção do pedido.
