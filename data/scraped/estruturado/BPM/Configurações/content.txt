            Configurações
            Para melhor usabilidade, é possível administrar e personalizar alguns processos e recursos dentro do BPM conforme for necessário. É importante lembrar que algumas configurações e ações só poderão ser realizadas por usuários administradores da plataforma. 
            As configurações padrão disponíveis dentro da plataforma BPM são:
            Configurar permissões
            
            Configurar SLA
            
            Fonte de dados externa
            
            Fonte de dados tabelas
            
            Formulários ECM
            
            Formulário Web Hospedado
            
            Interface WEB
            
            Mecanismo de atribuição
            
            Notificações
            
            Serviços externos
            
            Visibilidade das variáveis
            
            Proprietário do processo
            
            Variáveis do processo
