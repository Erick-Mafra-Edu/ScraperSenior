            Terminal de autoatendimento com acesso remoto (tablet)
            O terminal de autoatendimento está disponível para uso em dispositivos móveis (tablet).
            Como instalar o aplicativo para ter essa versão do terminal de autoatendimento?
            
                Na Senior X, em Ronda Senior > Ronda senior X > Downloads, baixe o arquivo Aplicação de Autoatendimento e depois extraia-o.

                Descompacte o arquivo	no servidor desejado.
                Configure o arquivo application.properties, incluindo as informações do local de acesso do aplicativo, de acordo com o exemplo disponível no próprio arquivo, para o produto desejado.
			ImportanteSe for utilizar o autoatendimento em tablet, defina o endereço como https para que seja possível utilizar a câmera do dispositivo.
                Execute o arquivo create-service.bat.
                Inicie o serviço através do arquivo start-service.bat.
            
            
                Garanta que a Portaria Avançada está configurada.
                No SGU, crie um grupo chamado TERMINAL e associe o usuário que será logado no terminal a este grupo.
                Ao fazer o cadastro do catálogo do dispositivo, defina para o campo Tipo do fabricante o valor "Acesso remoto"
            
            
                Cadastre um dispositivo do tipo Terminal Autoatendimento e cujo Fabricante seja "31 - Acesso Remoto".
            
            Como começar a usar o autoatendimento em tablet?
            Através do navegador, acesse o servidor onde foi instalada a aplicação do terminal com a porta configurada no arquivo application.properties  e o Terminal de Autoatendimento está pronto para uso.
            Utilizando dispositivo móvel, quais as diferenças em relação ao Terminal de Autoatendimento?
            Em termos de funcionalidade, todas as disponíveis no tablet se equiparam ao terminal. Entretanto:
            
                A leitura do QR Code de identificação de agendamento é feita pelo leitor do dispositivo.
                O software não fica embarcado dentro do próprio hardware, pode estar em qualquer servidor, ao qual o tablet é conectado.
                O tablet não tem o dispenser para entregar o crachá, mas imprime uma etiqueta que serve como crachá de identificação (utiliza a impressora padrão configurada no próprio tablet, não é uma configuração do sistema Ronda Senior).
