            FAQ - Gerais
            Como seleciono uma imagem como 
 papel de parede no sistema?
                
                    O arquivo de imagem a ser utilizado como papel de parede pode ser configurado na Central de Configurações Senior (SeniorConfigCenter.exe, que se localiza no diretório onde o sistema está instalado). Acesse o caminho Sistemas > Ronda Senior > Configurações e então selecione a imagem desejada no campo Arquivo de papel de parede.
                
            
            Qual a chave utilizada na Central de Configuração Senior para utilização das leitoras Smart?
                
                    Existe o campo Local do arquivo chave para leitura de crachás SmartCard Telemática na Central de Configurações Senior. Esta chave contém 
 o diretório do arquivo CHAVE.CFG utilizada na rotina de Smart Card. Na
 tela de leitora ativa do usuário foi retirado o campo no qual era informado 
 o diretório do arquivo chave.cfg.
                
            
            O que é um Coletor?  
                
                    É um equipamento dotado de dispositivo mecânico para liberação ou 
 bloqueio físico de passagem, controlado por um circuito eletrônico baseado 
 em microprocessador. Existem diversos tipos de dispositivos, tais como 
 catracas, torniquetes, cancelas, fechaduras eletrônicas e dispositivos biométricos.
                    Os dispositivos são equipamentos 
 com EPROM (memória não volátil, que não se perde quando cai a energia 
 elétrica). Esta memória é utilizada quando o dispositivo perde a comunicação com 
 o restante do sistema. Nesta situação, o dispositivo tem autonomia para 
 assumir o controle do bloqueio mecânico para liberar ou 
 bloquear o acesso em modo Off-line.
                
            
            O que é o Código Alfa?  
                
                    Este código identifica fisicamente o dispositivo, pois o código cadastrado 
 no sistema não é igual ao código físico dele. Este código deve ser passado 
 pela Telemática, pois é através dele que os comandos são enviados.
                
            
            O que é Gateway?
                
                    É um microcomputador dedicado que realiza a interface entre os dispositivos 
 e a base de dados de controle do Acesso. Cada centralizadora pode ter 
 12 dispositivos conectados. Atualmente a Telemática já possui um equipamento 
 chamado centralizadora que faz a função do microcomputador chamado de 
 ARLON.
                
            
            O que é o Aplicativo do Coletor?
                
                    Este aplicativo é gerado pelo software da Telemática chamado de PGRWIN, 
 onde deve-se configurar as características e os comportamentos dos dispositivos, 
 com por exemplo: Antidupla, função, acionamento, criptografia, níveis 
 de acesso, etc.
                
            
            O que é a Central de Configurações Senior?  
                
                    É um aplicativo utilizado para editar os arquivos de configuração 
 da Senior. Estes arquivos contém os endereços de diretórios, arquivos, 
 e conexão do banco, pois o aplicativo utiliza este arquivo para a inicialização.
                
            
            Como pode-se criar usuário no sistema?  
                
                    Para criar usuário no sistema, deve-se utilizar o aplicativo SGU (Sistema 
 Gerenciador de Usuário) que foi criado com esta finalidade. Para isto 
 é necessário ter a senha do usuário master do sistema ou da senha do usuário 
 do banco.

                        Altere para a versão do cliente;

            Como funciona a criptografia de crachás quando utilizado equipamentos 
 Telemática?
                
                    Se o cliente utilizar criptografia deve-se setar em definições do crachá 
 o campo Criptografa Crachá como "S - Sim". O padrão 
 de criptografia é o mesmo para todos os clientes, utilizando as DLLs CVT32.dll 
 e DCVT32.dll de criptografia e descriptografia respectivamente. No entanto 
 o algoritmo de criptografia é exclusivo de cada cliente, utilizando o 
 arquivo cliente.cfg, disponibilizado pela Telemática, que por sua vez 
 deve ser apontado na chave Local padrão do arquivo chave para leitura de crachás Smart Card Telemática na Central de Configurações Senior.
                
            
            Quais são os tipos 
 de leitoras de crachás suportadas pelo sistema?  
                
                    O sistema suporta qualquer leitora padrão Windows que não seja de 
 teclado, pois este modelo o sistema não tem como identificar se a informação 
 vem do teclado ou da leitora.
                
            
            Como importar todas as fotos dos colaboradores?  
                
                    Para que isso seja possível é necessário que haja um diretório/pasta 
 com as fotos dos colaboradores cujo o nome do arquivo deva ser o número 
 do cadastro do mesmo. Constado a existência destas fotos, acesse Colaboradores > Fotos e mande importar todas as fotos.
                
            
            Para que serve o cadastro de separação 
 de marcações?  
                
                    Esta tela serve para os dispositivos, suas funções e demais identificações 
 da marcação e tem como objetivo definir como o sistema deverá tratar a marcação.
                    Exemplo:
                        
                            Dispositivo 001 – Função 
 001 – Solicitação de Vale Transporte
                            Dispositivo 001 – Função 
 002 – Marcação de Ponto
                        
                    
                
            
            Que modelo o sistema utiliza quando o usuário clica no botão imprimir 
 nas telas de crachás?  
                
                    Utilizam os modelos configurados em Definições de Acesso > Modelos. 
 Nesta tela deve-se configurar os modelos que serão utilizados na impressão 
 dos crachás.
                
            
            Qual configuração de banco de dados Oracle o sistema utiliza para inserir valores no banco?
                
                    Os sistemas da Senior utilizam o charset do banco de dados como "Cp1252".
                    A diferença é o tamanho utilizado para armazenar caracteres diferenciados.
                    Exemplo:
                        
                            
                                A palavra "Não" em UTF-8 ocupa 4 bytes.
                                A palavra "Não" em Cp1252 ocupa 3 bytes.
                            
                            Para esta configuração o campo NLS_CARACTERSET da tabela v$nls_parameters deve estar com o valor WE8MSWIN1252.
                        
                    
                
            
            Quais são as portas de serviços que o sistema utiliza?
                
                    O Manual de Instalação contém a documentação das Portas de serviços.
