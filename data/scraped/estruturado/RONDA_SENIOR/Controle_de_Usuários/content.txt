            FAQ - Controle de 
		 Usuários
            Como utilizar o logon de usuário utilizando o sistema operacional?
                
                    Caso seja utilizada esta funcionalidade, ao entrar no 
 Acesso WEB, o sistema não apresentará a tela para informar o usuário e 
 senha, passando diretamente para a tela principal do sistema.
                    A configuração deste recurso está no SGU (Sistema Gerenciador 
 de Usuários). Acessando a tela Segurança > Logon com usuário do Sistema 
 Operacional, deve-se selecionar o assinalamento Permitir logon 
 com o usuário do Sistema Operacional. Após selecionado este assinalamento, 
 deve-se configurar quais usuários terão permissão para fazer logon com 
 a senha do Windows.
                    A atribuição da permissão para logar automaticamente 
 com o usuário do sistema operacional segue os padrões das permissões já 
 existentes, e é cadastrada a partir do menu Segurança > Direitos 
 Gerais do aplicativo SGU. Selecionando o item Efetuar o Login 
 com o usuário do Sistema Operacional será possível adicionar os 
 usuários com permissão no painel abaixo. Posteriormente, deve-se configurar 
 a tela de Formas de logon aos sistemas por tipo de acesso 
 com a opção Logon com usuário do Sistema Operacional. 
                    
                        Observações
                        Os arquivos ActiveLogon.cab e sppauto.htm, que encontram-se na cópia, 
 deverão estar no diretório virtual. Além disso deve-se alterar a página inicial 
 do servidor de páginas pelo arquivo sppauto.htm. Esse processo 
 pode ser realizado através do Internet Information Services no acesso 
 pelo painel de controle. Após feito o acesso, abra a árvore dos HTMLs 
 acessando o tema acesso. Acessar pelo botão direito do mouse as propriedades 
 na pasta documentos substituir pelo arquivo sppauto.htm disponível na 
 cópia.
