            Criptografia de crachá de código de barras
            Configurar dispositivos
            Inicialmente é necessário definir se os crachás de código de barras utilizam ou não criptografia. Para isso, acesse a tela Definições do Crachá, guia Gerais, e selecione a opção desejada no campo Criptografar Crachá.
            Caso escolha não utilizar criptografia nos crachás, basta enviar o comando de Carga de Aplicativo.
            Utilizando Criptografia Senior: Para que os dispositivos utilizem a criptografia definida pela Senior, utilizando como chave o código do cliente obtido na proprietária, deve-se acessar a Central de Configurações Senior e preencher a chave DLLCodigoBarras (Sistemas > Ronda Senior > Geral > Arquivo DLL para criptografia e descriptografia de crachás Senior), sendo esta do tipo Arquivo e apontando para a DLL SPBarCode.dll. Desta forma, ao enviar o comando de Carga de Aplicativo, o dispositivo será configurado para descriptografar o crachá utilizando o algoritmo Senior.
            Utilizando Criptografia Telemática: Para que os dispositivos utilizem a criptografia definida pela Telemática, utilizando como chave o código do cliente obtido no arquivo Cliente.cfg, deve-se acessar a Central de Configurações Senior e preencher as chaves DLLCriCra e DllDesCra (Sistemas > Ronda Senior > Geral > Arquivo DLL para criptografia/descriptografia de crachás Telemática), sendo esta do tipo Arquivo e apontando para a DLL Cvt32.dll. Desta forma, ao enviar o comando de Carga de Aplicativo, o dispositivo será configurado para descriptografar o crachá utilizando o algoritmo Telemática.
            Efetuar leitura de código de barras
            Se uma leitora de código de barras for utilizada e a criptografia de crachá estiver configurada (independente de Senior ou Telemática), um botão com ícone de raio irá aparecer ao lado dos campos de crachá. Ele ativa leitoras de crachá, para que a leitura possa ser efetuada.
            Ao clicar no botão, se a leitora ativa for do tipo barras, uma janela será aberta indicando ao usuário que ele deve efetuar a leitura do crachá neste momento. Quando o valor numérico for lido ele será descriptografado, a janela será fechada e o campo será preenchido com o número correto do crachá, após a descriptografia.
            O usuário também pode, opcionalmente, apenas digitar o número criptografado do crachá no campo utilizado para realizar a leitura. 
            
                Observação
                
                A leitura de crachá com descriptografia somente funciona com o sistema on-line. Se ele estiver off-line, o número do crachá deve ser digitado.
