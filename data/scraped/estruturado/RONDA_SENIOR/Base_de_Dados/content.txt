            FAQ - Base de Dados
            Quais são os bancos suportados pelas 
 aplicações da Senior?
                
                    Atualmente estas soluções estão disponíveis para os SGBD (Sistemas 
 Gerenciadores de Bancos de Dados) indicados na página de Plataformas Homologadas do site da Senior. Esta página sempre apresenta as versões mais atualizadas dos bancos suportados por estas aplicações.
                
            
            O que é o TBS ?  
                
                    Este arquivo contém as definições do banco de dados como:
                    
                        
                            Tabelas;
                            Stored Procedures;
                            Triggers;
                            Views;
                            Listas;
                        
                    
                    TBS é a abreviação de 
 tabelas. Este arquivos somente pode ser aberto pelo aplicativo CBDS.exe, 
 também pertencente à Senior.
                
            
            O que é o CBDS?
                
                    É um aplicativo utilizado como dicionário de dados. Ele apresenta 
 vários recursos como: gerar base, backup, usuário no banco, consistência 
 de base e outros recursos referente ao banco de dados.
                
            
            O que é a consistência 
 de base?
                
                    É um recurso que deve ser utilizado quando a versão do TBS é alterada. Isto significa que foram incluídos ou excluídos campos, tabelas, listas, 
 stored procedures ou views, e quando ocorre esse tipo de mudança por algum motivo, a 
 base do cliente deve ser atualizada com esta alteração. Em outras palavras, o CBDS compara o que há no TBS com o 
 que consta na base de dados, atualizando a base.
                
            
            O que devo configurar a mais para funcionar o acesso on-line 
 em banco MS-SQL Server?
                
                    Em definições de acesso deve-se setar o usuário Telemática, pois para os 
 comandos neste banco é necessário colocar o nome do dono das tabelas na 
 frente da tabela conforme mostra o exemplo abaixo:
                    
                        SELECT * FROM TELESSVR.CONTROLE
