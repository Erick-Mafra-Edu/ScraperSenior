            FAQ - Veículos
            Qual a diferença entre usuários e condutores da frota?  
                
                    Os condutores não possuem veículo fixo de forma que podem agendar para qualquer um que tenha permissão para conduzir. O usuário é a pessoa que recebe um veículo fixo para trabalhar, e para trocar de veículo é necessário utilizar a tela de troca.
                
            
            O que é um veículo de teste?  
                
                    Veículos que são utilizados pelas montadoras para testar alguma nova tecnologia. Este veículo recebe algumas licenças especiais.
                
            
            O que é DUT?  
                
                    Nota Fiscal/Licença normal para a circulação do veículo, este recurso é utilizado nos veículos de testes.
                
            
            O que é o romaneio?  
                
                    Formulário que especifica os equipamentos a serem testados como: injeção eletrônica, bomba de óleo, etc.
                
            
            O que é comodato?
                
                    Licença originária do orgão público (nota fiscal série 19) para que o veículo teste possa se movimentar em via urbana. Normalmente o prazo especificado é de trinta dias.
                
            
            O que é dotação?
                
                    É quando o veículo de testes contém todos os ítens obrigatórios de segurança.
                
            
            O que é AIT?
                
                    Auto de Infração de Trânsito. É um dos meios legais de se registrar uma infração. O AIT deve conter os dados mínimos necessários, conforme art. 280 do CTB e legislação complementar.
                
            
            O que significa CTB?
                
                    Código de Trânsito Brasileiro.
                
            
            Existem listas para veículos?
                
                    Não há listas de bloqueio e liberação para veículos.
                
            
            Como funciona o controle anti-dupla com veículos?
                
                    Os crachás de veículos utilizam as validações do condutor, somente o crachá do veículo não participa de tipo algum de validação.
