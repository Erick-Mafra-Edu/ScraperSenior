            Web services do Ronda Senior
            Para utilizar os web services, é obrigatório o preenchimento dos campos Usuário, Senha e Encryption. Essa validação garante a segurança de dados sigilosos, pois impede que usuários não autorizados realizem requisições de dados sem nenhum controle.
            Nos campos Usuário e Senha, informe os dados de um usuário com acesso às aplicações e com permissão para executar os webservices.
            Já no campo Encryption, preencha  0 (zero), para indicar que os valores serão enviados em texto, ou 1, para indicar que os dados serão criptografados. 
            A seguir, a lista de web services disponíveis:
            
                
                    
                        Web service
                        Descrição
                    
                    
                        com.senior.g5.sp.acc.portal.pessoas.presentes
                        
                        
                            Retorna a quantidade de pessoas presentes por local físico, contabilizando tipo de pessoa e quantidade, ou verifica se uma pessoa específica está presente em um determinado local físico. 
                        
                    
                    
                        com.senior.g5.sp.acc.registra.evento
                        
                        Registra eventos de acesso (tabela R070ACC).
                    
                    
                        com.senior.g5.sp.aluno.atividades
                        
                         Permite cadastrar e alterar as atividades dos alunos, conforme a tela Aluno - Atividades (FR073ATA).
                    
                    
                        com.senior.g5.sp.aluno.disciplinas
                        
                        Este web service possui o mesmo comportamento da tela Aluno - Disciplina (R073DIA).
                    
                    
                        com.senior.g5.sp.aluno.historicocracha
                        
                        Este web service possui o mesmo comportamento da tela Histórico de Crachás de Alunos (FR073HCH).
                    
                    
                        com.senior.g5.sp.alunos.cadastro
                        
                        Este web service possui o mesmo comportamento da tela Alunos (FR073ALU).
                    
                    
                        com.senior.g5.sp.bio.carrega.biometria
                        
                        Determina quais as templates biométricas serão carregadas em memória para realizar a identificação biométrica no serviço "Ronda Senior - Identificação Biométrica Nitgen".
                    
                    
                        com.senior.g5.sp.bio.identifica.biometria
                        
                        Realiza a identificação biométrica do visitante, com base na template de biometria enviada.
                    
                    
                        com.senior.g5.sp.bio.remove.biometria
                        
                        Retira a template da utilização na identificação biométrica do serviço "Ronda Senior - Identificação Biométrica Nitgen".
                    
                    
                        com.senior.g5.sp.email
                        
                        Realiza o envio de e-mail.
                    
                    
                        com.senior.g5.sp.ger.portal.prop
                        
                        Obtém módulos definidos na proprietária. Módulos verificados: Acesso On-Line (AO), Portaria (PO), Frota (FR), CFTV (TV) e Alarmes (AL).
                    
                    
                        com.senior.g5.sp.lgpd.DeleteLGPDData
                        
                        
                            Serviço responsável por excluir da base ou anonimizar os dados pessoais do titular das informações. 
                            A exclusão/anonimização respeitará as definições do arquivo de mapeamento, ou seja, serão excluídas/anonimizadas apenas as informações que estiverem assim definidas.
                        
                    
                    
                        com.senior.g5.sp.lgpd.QueryLGPDData
                        Serviço responsável por buscar os dados pessoais do titular das informações, indicando se cada informação é sensível e/ou legal.
                    
                    
                        com.senior.g5.sp.pacientes.cadastro
                        
                        Este web service possui o mesmo comportamento da tela Cadastro de Paciente (FR097PAC).
                    
                    
                        com.senior.g5.sp.pacientes.visitas
                        
                        Web service para informações sobre as visitas dos pacientes.
                    
                    
                        com.senior.g5.sp.registroocorrencia.cadastro
                        
                        Permite a inclusão de incidentes de colaborador.
