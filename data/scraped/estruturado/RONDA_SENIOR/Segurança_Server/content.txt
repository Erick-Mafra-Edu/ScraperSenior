            Segurança Server
            No Segurança Server (Sserver) são executados vários processos que se 
 refletem diretamente no Ronda Senior. Este aplicativo tem 
 como objetivo gerenciar as requisições do usuário, sejam elas feitas pelo 
 acesso web, na integração com outros sistemas ou na execução de vários 
 outros processos automáticos.
            O Sserver é aplicado ao  Ronda Senior integrado e não integrado.
            Funcionalidades
                Dentre as principais funcionalidades do Sserver, podem-se destacar: 

            
                Segurança SID: Serviço 
 de Informações Dinâmicas. Para mais informações, consulte a documentação Ferramentas de Customizações > Integração entre Sistemas > Serviços de Informações Dinâmicas > Segurança - Serviço de Informações Dinâmicas;
                Segurança Web: interface 
 web para os usuários do Ronda Senior;
                Processos Automáticos:  o Sserver gerencia e executa as requisições dos usuários feitas através 
 de um processo automático. Encontre mais informações
 no sistema, em Recursos > Processos 
 Automáticos. 
            
            O SServer conta com um log avançado dos Processos Automáticos, que mostra 
 detalhadamente o consumo de memória de cada passo do processo. Para exibir 
 o log é necessário utilizar o parâmetro "-logproc".
            
                Importante
                
                Para o correto funcionamento do SServer:
                
                    Coloque o executável nas exceções do antivírus. Recomendamos que seja colocada toda a pasta Senior;
                    Rode o SServer com um usuário com privilégios administrativos;
                    Desabilite a UAC do Windows, evitando que a aplicação fique travada.
