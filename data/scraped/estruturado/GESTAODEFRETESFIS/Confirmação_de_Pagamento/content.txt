            Confirmação de Pagamento
            
                
                
                
                    
                        Descrição
                         
                    
                    
                        Url de serviço
                        
                        https://platform.senior.com.br/t/senior.com.br/bridge/1.0/rest/fis/integracao/actions/confirmacaoPagamento

                    
                    
                        Tabela
                        
                        TI_WS_E_CONFIRMACAO_PAGAMENTO
                    
                    
                        Tipo de envio
                        
                        POST
                    
                    
                        Momento
                        
                        Confirmação dos pagamentos
                    
                    
                        Array
                        
                        Sim- Poderão ser enviadas várias confirmações de pagamentos na mesma chamada.
                    
                
            
            
                
                
                
                    
                        Chave
                        
                        Valor
                        
                    
                    
                        Contente-Type
                        
                        Aplication/json
                    
                    
                        Authorization
                        
                        Bearer inserir_token_aqui
                    
                
            
            Detalhamento dos atributos JSON entrada
            
                
                
                
                
                
                
                    
                        Nome do Atributo
                        Tipo de Dado
                        Default
                        Obrigatório
                        Descrição
                    
                    
                        confirmacaoPagamentos

                        Array

                         
                        S
                        Recebe um array de objetos
                    
                    
                        pag_cnpj_empresa

                        String

                         
                        S
                        CNPJ da Empresa
                    
                    
                        pag_cd_cnpj_tranp

                        String

                         
                        S
                        CNPJ da trasnportadora
                    
                    
                        pag_nu_cte

                        String

                         
                        S
                        Número do CTE
                    
                    
                        pag_nu_cte_serie

                        String

                         
                        S
                        Série do CTE
                    
                    
                        pag_vlr_pago

                        String

                         
                        S
                        Valor total pago
                    
                    
                        pag_dt_pagamento

                        String

                         
                        S
                        Data do pagamento
