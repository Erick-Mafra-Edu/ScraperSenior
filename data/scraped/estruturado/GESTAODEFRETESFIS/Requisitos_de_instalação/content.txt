            Requisitos de instalação
            Para a instalação do WebService o cliente deve disponibilizar um servidor dedicado ao serviço com os seguintes pré-requisitos:
            •	Processador 2GHz com 2GB de RAM, ou superior. Se a máquina for compartilhada com outros serviços, sugerimos uma configuração melhor;
            •	Sistema operacional Linux ou Windows, de preferência Linux;
            •	Instalação da JDK 8 (Java);
            •	Instalação do servidor de aplicação web TomCat 7;
            •	Este servidor deve possuir acesso ao servidor de banco de dados;
            •	A Senior deve possuir acesso a este servidor para podermos realizar a instalação do sistema e posterior manutenção;
