            Financeiro
            Esse tópico  permite realizar a conferência das faturas que são enviadas pelas transportadoras para pagamento.
            Auditoria frete
            Problemas com a Tabela de Frete
                
                    Compara-se os serviços contratados e os valores contidos nas tabelas de frete para verificar se os dados estão coerentes e se as faturas podem ser liberadas para serem pagas.
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes > Financeiro > Auditoria Frete
                    Através de filtros é possivel escolher a situação dos pedidos
                        
                            Filtro de Faturado: 
                            
                                Sim 
                                Não 
                            
                            Filtro de Conferência:
                            
                            
                                Divergente - fatura com cálculo maior ou menor ao previsto pela transportadora.
                                Conferido - fatura com cálculo correto.
                            
                            
                                Limite - permite a diferença de valor da fatura, baseada no valor limite da transportadora. 
                            
                            Filtro Tipo: permite a seleção pelo cálculo de cobrança das transportadoras.
                            
                                (+) Maior
                                (-) Menor
                            
                            Filtro por Período: escolha o período desejado.
                        
                    
                
            
            Erro no Cálculo
                
                    Nesse caso, a cobrança do frete pode vir tanto maior quanto menor do que o correto, assim sendo, é possível fazer o reajuste das divergências com os valores correto para pagamento
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes > Financeiro > Auditoria Frete.
                    
                        Erro ocorrido pela transportadora: selecione a fatura divergente e use botão Aceite FIS.
                    
                    Opcionalmente é possivel realizar o aceite em lote, use o botão Aceite FIS.
                    
                        Erro ocorrido pelo Sistema: selecione a fatura divergente, use botão Aceite Transportadora.
                    
                    Opcionalmente é possivel realizar o aceite em lote, use o botão Aceite Todos Transp.
                
            
            Liberação Faturamento
            Esta funcionalidade permite verificar os pedidos aptos para liberação de faturamento.
                
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes> Financeiro > Liberação de Faturamento.
                    
                        Use o botão,Filtro por período, e informe o período para a consulta.
                        No botão, Notas Fiscais, é possivel listar e visualizar todas as notas fiscais emitidas.
                        Para realizar a liberação e faturamento de pedidos, use o botão Liberar. Após a  liberação  é enviado o arquivo prefat para a transportadora. Opcionalmente, é possivel fazer a liberação de pedidos dentro de uma mesma lista, use o botão Liberar Todos e informe a data da fatura.
                        Caso deseje desfazer o aceite realizado na Tela de Auditoria de Frete, use o botão Desfazer Aceite
                    
                
            
            Quais Consultas/Pesquisas Posso Realizar?
            Consulta Faturas
                
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes FIS > Financeiro > Consulta Faturas
                    Permite verificar o retorno com os dados de faturamento por meio da integração com o ERP, e os nomes dos arquivos PREFAT gerados para as transportadoras.
                
            
            Consulta CTe
                
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes FIS > Financeiro > Consulta CTe
                    Permite verificar o retorno com a validação da chave de acesso do CTe.
                
            
            Consulta Geral de Documentos de Roubo/Extravio
                
                    Acesse: XPlatform > Gestão Logística > Gestão de Fretes FIS > Financeiro > Consulta Geral de Documentos de Roubo/Extravio
                
            
            Documentos Não Pagos
                
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes FIS > Financeiro > Documentos Não Pagos
                    Pesquisa os documentos faturados, cujos arquivos de integração de confirmação de pagamento já tenham sido importados informando assim o seu status.
                
            
            Provisionamento de Documentos
                
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes FIS > Financeiro > Provisionamento Documento
                    Permite verificar os documentos calculados e romaneios gerados, possibilitando uma análise dentro do período.
                
            
            Integração de confirmação de Pagamento
                
                    Acesse: XPlatform> Gestão Logística > Gestão de Fretes FIS > Financeiro > Integração de Confirmação de Pagamento
                    Esta aplicação importa o arquivo com as informações de pagamento da fatura.
                
            
             Troca de informações via EDI
             Nesta etapa que envolve  auditoria de frete e envio de fatura, a troca de mensagens é feita assim:
            
                A transportadora envia um arquivo OCOREN para o embarcador, sinalizando que a entrega foi realizada;
            
            
                O embarcador faz uma análise das informações recebidas e gera uma pré-fatura, enviando o arquivo PREFAT para a transportadora, informando quais CT-es estão liberados para que a cobrança seja efetuada;
            
            
                A transportadora gera uma fatura de cobrança para o embarcador, baseando-se nos documentos que já foram auditados e liberados previamente. 
            
            
                Nota
                Para mais informações utilize o tópico de Intercâmbio Eletrônico de Dados
