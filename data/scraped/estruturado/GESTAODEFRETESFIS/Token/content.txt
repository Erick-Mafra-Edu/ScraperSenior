            Token
            
                
                
                
                    
                        Descrição
                        
                         
                    
                    
                        Url do Serviço
                        
                        https://platform.senior.com.br/t/senior.com.br/bridge/1.0/rest/platform/authentication/actions/login

                    
                    
                        Tipo de envio
                        
                        POST
                    
                    
                        Momento
                        
                        Utilizado para retornar o token de validação das chamadas de integração na plataforma
                    
                    
                        Observação
                        
                        A requisição do token deve ser feita anteriormente as demais chamadas de integração e as demais chamadas deverão conter no header o valor do access_token que será retornado como mostra o Json de saida abaixo.

                    
                    
                        Array
                        
                        Não
                    
                    
                        Header
                        
                        Chave: Content Type valor: Application/json
                    
                
            
            
                
                
                    
                        Exemplo do layout de entrada em JSON
                    
                    
                        
                            {
                            "username":"admin@empresa.com.br",
                            "password":"senhaUsuario"
                            }
                        
                    
                
            
            
                
                
                    
                        Exemplo do layout de saída em JSON
                    
                    
                        
                            {
                            "jsonToken": {
                            "scope":"default",
                            "expires_in":85846,
                            "username":"admin@cempresa.com.br",
                            "token_type":"Bearer",
                            "access_token":"482ee34ec1ba44b73eb1c2b6f7da6c55",
                            "refresh_token":"95b9f124761a18e7b2f62affbf311764"
                            }
                        
                    
                
            
            Regra
                
                Primeiro é necessário realizar a requisição do Token para só assim poder usufruir dos outros serviços de integração.
