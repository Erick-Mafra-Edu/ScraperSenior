            Instalação do serviço SRVPORTALCLIENTE 
                
                    Este serviço utiliza o RabbitMQ para realizar o envio e o recebimento das mensagens. Para isso, é necessário acessar as variáveis de sistema, e adiciona as seguintes chaves:
                    
                        BROKER_HOST: O valor será o endereço do RabbitMQ seguido da porta 5672, ex: 10.8.0.134:5672
                        BROKER_USER: Usuário para acessar o RabbitMQ, o padrão é admin
                        BROKER_PASS: Senha para acessar o RabbitMQ, o padrão é admin
                        VIRUTAL_HOST: Virtual host que será criada a fila no RabbitMQ, o padrão é /
