            Instalação do serviço de Integração FMS
            Primeiros Passos: 
            Primeiro passo é realizar o passo a passo do Guia de Implantação. Após realize a instalação e configuração do serviço de integração com o FMS.
            Instalação do serviço Fmsintegrador
                
                    Proceda com um dos comandos abaixo, utilizando o usuário root da máquina virtual Linux:
                    Comando de exemplo de instalação/configuração no servidor Linux para conexão através do Banco de Dados Oracle, apenas alterar o que está em negrito e caixa alta:
                    docker run -d -e BROKER_HOST=IP_LOCAL -e SOFTRAN=/root/ -e DB_TYPE=Oracle -e TENANT=TENANT -e JAVA_OPTS="-Dspring.profiles.active=dev -Ddb.url=jdbc:oracle:thin:@//IP_BD:1521/INSTANCIA_BD -Ddb.username=USUÁRIO_BD -Ddb.password=SENHA_BD -Duser.timezone=America/Sao_Paulo -Dservicemodel.logger.br.com.senior=INFO" --name=fmsintegrador seniorsa/fmsintegrador:LATEST
                    Comando de exemplo de instalação/configuração no servidor Linux para conexão através do Banco de Dados MSSQL, apenas alterar o que está em negrito e caixa alta:
                    docker run -d -e BROKER_HOST=IP_LOCAL -e SOFTRAN=/root/ -e DB_TYPE=SqlServerJtds -e TENANT=TENANT -e JAVA_OPTS="-Dspring.profiles.active=dev -Ddb.url=jdbc:jtds:sqlserver://IP_BD:1433/INSTANCIA_BD -Ddb.username=USUÁRIO_BD -Ddb.password=SENHA_BD -Duser.timezone=America/Sao_Paulo -Dservicemodel.logger.br.com.senior=INFO" --name=fmsintegrador seniorsa/fmsintegrador:LATEST
                    Onde:
                    
                        IP_LOCAL: IP do próprio servidor Linux;
                        TENANT: Tenant do cliente;
                        IP_BD: IP do banco de dados utilizado pelo Sistema TMS Senior;
                        INSTANCIA_BD: Nome da Instância do Banco de Dados (obter informação com DBA do cliente);
                        USUÁRIO_BD: Usuário do Banco de Dados;
                        SENHA_BD: Senha do Banco de Dados.
                        LATEST: Última versão do serviço
                    
                
            
            Configurações Senior X
            Croqui: 
                
                    Criação do agendamento
                        
                            Para que as informações sejam continuamente integradas,é necessário a criação do agendamento. Para criar um agendamento, acesse a Senior X, menu Tecnologia > Customização > Agendamentos. Procure pelo o domínio frota e criar os agendamentos no serviço fmsintegrador.
                        
                    
                    Agendamento de integração do Croqui: 
                        
                            O agendamento tem como função, integrar os dados referentes aos Cadastros de Empresa, Funcionário, Veículo e Motorista vinculado com o Veículo e as movimentações de Entrada e Saída.
                            Dados do Agendamento
                            
                                Nome do agendamento: integração croqui
                                Situação: Ativo
                                Funcionalidade: frotaSincronizaCroqui
                                Parâmetro de entrada: JSON
                                Entrada: {}
                            
                            Execução
                            
                            
                                Iniciar em: data atual
                                Horário: horário atual + 5 minutos
                                Repetir: Marcado
                                Repete a cada: minuto | divisível por | 1
                                Termina em: Nunca
                            
                        
                    
                    Agendamento de integração do Croqui - cadastro checklist
                        
                            Tem como função, integrar os dados referentes ao cadastro de Checklist.
                            Dados do Agendamento
                            
                            
                                Nome do agendamento: integração croqui - cadastro checklist
                                Situação: Ativo
                                Funcionalidade: checklistSincronizaCroqui
                                Parâmetro de entrada: JSON
                                Entrada: {}
                            
                            Execução
                            
                            
                                Iniciar em: data atual
                                Horário: horário atual + 5 minutos
                                Repetir: Marcado
                                Repete a cada: minuto | divisível por | 1
                                Termina em: Nunca
