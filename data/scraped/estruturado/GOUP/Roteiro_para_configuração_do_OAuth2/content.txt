            Teclas de atalho
            O sistema dispõe de teclas de atalho para facilitar a navegação e usabilidade dos recursos disponíveis. Confira:
            Atalhos gerais
            
                F1: exibe o Manual do Usuário;
                F3: mostra as opções disponíveis nos campos de buscas ();
                Alt + F4: fecha a solução ERP;
                Ctrl + F4: fecha somente a tela aberta;
                F5: seleciona outra empresa;
                F6: seleciona outra filial;
                F7: altera o usuário logado no sistema;
                F8: mostra as tabelas e campos do sistema;
                F9: seleciona relatórios gerados para visualização;
                F11: acesso rápido para as telas;
                F12: exibe a calculadora;
                Alt: exibe a barra de menu. Este atalho não está disponível para ser utilizado no GO UP XT quando com o seu menu estiver habilitado. Desta maneira, caso o menu do XTended estiver desabilitado, será possível utilizá-lo;
                Ctrl + F executa a busca de componentes, como nome de campos, guias e grades. Lembrando que esse atalho não localiza o conteúdo dos componentes. Exemplo: 
            
            Atalhos de execução
            
                Ctrl + E: executa o recurso de envio de e-mail;
                Ctrl + M: executa a tela de consulta de registros multimoeda (F031CMM);
                Ctrl + P: executa a tela de visualização dos parâmetros de projetos (F615VPP);
                Ctrl + R: executa o gerador de relatório.
            
            Atalhos de navegação
            
                Tab: navega entre os campos da tela;
                Ctrl + Tab: navega para a  tela da direito (quando houver mais de uma tela aberta);
                Ctrl + Shift + Tab: navega para a  tela anterior (quando houver mais de uma tela aberta);
                Ctrl + Page Up: se houver guias dentro da tela selecionada, navega para a próxima guia (guia a direita);
                Ctrl + Page Down: se houver guias dentro da tela selecionada, navega para a guia anterior (guia a esquerda);
                Ctrl + F4: fecha a tela selecionada.
            
            
                Importante
                
                Essas teclas de atalho não possuem funcionalidade para telas SGI (Senior Gerenciador de Interfaces).
