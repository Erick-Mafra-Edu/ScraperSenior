                Consistências a Nível de Tabela
                As consistências deste nível, devem ser configuradas na própria tela de configuração da tabela.
                É possível consistir as permissões de inserção, alteração e exclusão de registros na tabela, pois as vezes estas ações devem ser disponíveis em casos específicos. Também é possível  criar um filtro para tornar a pesquisa mais fácil.  Para criar estas consistências, as condições devem ser preenchidas com expressões SQL correspondentes.
                Nota
                     Muitas vezes as tabelas do sistema não nos dão essa liberdade, pois o SGI por vezes restringe essa configuração somente as tabelas do usuário (cuja nomenclatura inicia em "USU_xxx").
                
                Filtro
                    Expressão SQL. Condição usada para determinar um filtro para pesquisar registros.
                Pode-se aplicar um filtro para cada tabela da Interface.  No filtro, é possível usar qualquer campo de qualquer tabela, desde que sejam tabelas da mesma Interface. Um campo da Interface, que não pertença a nenhuma tabela, não pode ser usado no filtro.
                Condição para ativar a inserção de Registros
                    Expressão SQL. Condição usada para determinar quando se pode inserir registros.
                Condição para ativar a alteração do registro corrente
                    Expressão SQL. Condição usada para determinar quando se pode alterar registros.
                Condição de validação geral
                    Expressão SQL. Condição usada para validar todas as operações da tabela.
                Condição para validar um novo registro
                    Expressão SQL. Condição usada para determinar quando um registro novo é valido.
                Condição para validar um Registro sendo alterado
                    Expressão SQL. Condição usada para determinar quando uma alteração é válida.
                Condição para ativar a exclusão do registro corrente
                    Expressão SQL. Condição usada para determinar a exclusão de um registro corrente.
