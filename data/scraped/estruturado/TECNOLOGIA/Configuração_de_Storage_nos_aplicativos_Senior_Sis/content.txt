            Configuração de Storage nos Aplicativos Senior Sistemas
            É de responsabilidade do cliente (DBA responsável pelo banco de dados) configurar a parametrização de storage no banco de dados e/ou aplicativos Senior Sistemas, por exemplo, o  CBDS. A configuração dessas opções otimiza os recursos do banco e melhora a performance. 
            As configurações de storage serão mantidas ao processar uma consistência, atualização e conversão de base de dados (operação de criar tabela/índice em quatro passos). Para os novos objetos, tabelas e índices será utilizada as parametrizações informadas no CFG.
            Informação de storage lidas do banco de dados
            Oracle
            Tabela:
            
            
                PCTUSED
                PCTFREE
                INITTRANS
                MAXTRANS
                TABLESPACE
                INITIAL
                NEXT
                MINEXTENTS
                MAXEXTENTS
                PCTINCREASE
                FREELISTS
                FREELIST GROUPS
                COMPRESSION:NOCOMPRESSCOMPRESSCOMPRESS FOR ALL OPERATIONS
                PARTITIONING:PARTITION BY RANGEINTERVAL:NUMTOYMINTERVALNUMTODSINTERVALSUBPARTITION BY RANGE
            
            Índice:
            
            
                PCTFREE
                INITTRANS
                MAXTRANS
                TABLESPACE
                INITIAL
                NEXT
                MINEXTENTS
                MAXEXTENTS
                PCTINCREASE
                FREELISTS
                FREELIST GROUPS
            
            MSSQL
            Tabela:
            
            
                TEXTIMAGE_ON
                FILE GROUP
            
            Índice:
            
            
                CLUSTERED/NONCLUSTERED
                FILLFACTOR
                FILE GROUP
            
            Parâmetros de configuração de storage disponíveis CFG
            Oracle
            
                DATA TABLESPACE
                INDEX TABLESPACE
            
            MSSQL Server
            
                DATA FILEGROUP     
                TEXTIMAGE FILEGROUP (somente na versão 5.2 da svcl)
                INDEX FILEGROUP
            
            CBDS - Recursos Especiais p/ ORACLE
            
                Configurações de armazenamento;
                Cadastro de informações de Storage (disponível somente na versão 5.2 da SVCL).
