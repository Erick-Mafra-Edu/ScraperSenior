            Tecnologia > Ferramentas de Apoio > Diagnóstico de Ambiente (Robô) > Executor > Opções
            
            
                
                    
                    
                    
                    
                    
                    
                
            
            Opções
            Este menu possibilita que você configure o diretório de onde os modelos 
 serão buscados ou visualizar o histórico de execuções. 
            
                
            
            Configurações...
            Esta tela permite configurar o diretório de onde os modelos serão buscados 
 pelo executor. Mesmo assim, ao executar um modelo, a ferramenta permite 
 que outros diretórios sejam utilizados.
            
                
            
            Diretório padrão para os modelos: 
 É possível incluir o caminho manualmente, digitando diretamente no campo 
 'Diretório padrão para os modelos', 
 ou então usar o botão  
 e selecionar de um diretório do sistema operacional:
            
                
            
            Ao clicar no botão 'Ok', o diretório 
 selecionado será preenchido no campo 'Diretório padrão para os modelos' 
 da tela anterior.
            Configurações Gerais
            Gerar Plano de Ação como: Essas 
 opções permitem configurar os dois tipos possíveis para gerar um Plano 
 de Ação, sendo eles:
            
                Pasta de Trabalho do Excel (*.xlsx): 
	 Planilha do Excel, compatível com a versão superior a 2003.
                Pasta de Trabalho do Excel 97-2003 
	 (*.xls): Planilha do Excel, compatível com as versões entre 
	 97 e 2003.
            
            Registro por 
 Páginas de Resultados: Ao configurar essa opção, a exibição das 
 ocorrências da ação após a execução do modelo será feita conforme a quantidade 
 informada aqui. As páginas serão quebradas, permitindo a navegação entre 
 uma e outra. Neste campo, são permitidos números entre 1 e 100.000.
            Histórico das execuções...
            Esta tela permite consultar as informações de todas as execuções de 
 modelos já realizadas. Além de utilizar os Filtros, é possível ordenar 
 os registros em ordem crescente ou decrescente, clicando sobre a coluna 
 correspondente.
            
                
            
            Filtro
            Data inicial: Permite informar 
 manualmente ou selecionar usando a tela de calendário do sistema, a partir 
 de qual data será iniciada a pesquisa de execuções.
            Data final: Permite informar 
 manualmente ou selecionar usando a tela de calendário do sistema, até 
 qual data será feita a pesquisa de execuções.
            Botão Aplicar: Ao abrir a tela, 
 são apresentadas todas as execuções já feitas, mesmo que os filtros estejam 
 informados, é preciso utilizar esse botão para que o filtro seja aplicado.
            Botão Limpar: Este botão limpa 
 o filtro aplicado restaurando todos os históricos de execuções (idêntico 
 como quando foi aberta a tela)
            Botão Ok: Fecha a tela, não 
 executa nenhuma ação.
            Botão Ajuda: Traz a documentação 
 do aplicativo.
