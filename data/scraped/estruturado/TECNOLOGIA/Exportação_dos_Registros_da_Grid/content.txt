                Exportação dos Registros da Grid
                As grades permitem a exportação dos registros exibidos para um arquivo 
 no formato XLSX, reconhecidos a partir da versão 2007 do Excel. Para exportar, clique com o botão direito sobre a grade e selecione a opção Exportar dados para arquivo .xlsx, em seguida indique o local e o nome do arquivo que será gerado com os dados exportados.
                Considerações 
 sobre o arquivo XLSX gerado
                
                    Acesso via Browser Access utilizando o navegador Safari com versão 
 4.0 em diante: Ao fazer o download de um arquivo com extensão XLSX utilizando o 
 navegador Safari 4.0 em diante, quando o sistema é acessado via Browser 
 Access, existe um comportamento padrão deste navegador que acrescenta 
 a extensão ZIP ao arquivo e o salva neste formato. Para solucionar esta 
 situação, basta retirar a extensão ZIP do nome do arquivo;
                    Datas "00/00/0000" na grade  serão exportadas como Texto e não como Data;
                    Valores nulos na base de dados serão exportados considerando as suas máscaras. Por exemplo, um valor nulo com a máscara do campo ZZ9,99, será exportado como "0,00".
                
                
                    Importante
                    
                    A exportação de dados em grids são exportadas apenas para o formato XLSX, que segue as as especificações do Excel (Microsoft). É possível utilizar ferramentas gratuitas, como o Libre Office, para a leitura de arquivos deste tipo de extensão, porém, elas podem apresentar algumas limitações próprias, principalmente quando se tratar do formato XLSX, que que é mais complexo que o XLS, pois permite mais tipos de formatação. Desta maneira, recomendamos que o arquivo seja aberto com Microsoft Office Excel.
