            Coleta de estatística em Oracle
            A estatística é um conjunto detalhado de informações coletadas do banco de dados, que são utilizadas para potencializar consultas. Com esses dados, o banco de dados Oracle define o melhor plano de execução para realizar as pesquisas.
            Rotinas de coleta
            Recomendação de rotinas de coleta para serem aplicadas aos cenários de implantação do ambiente.
            Esta rotina deve ser agendada e executada por um DBA e, por recomendação, no mínimo uma vez por semana. Caso agendada, desative a rotina de coleta automática do banco, Auto Optimizer Stats Collection, ou bloqueie as estatísticas da base que será feita a coleta, para não ser sobreposta.
            Dicionário de dados e sistema em Oracle
                
                    
                        begin
                        DBMS_STATS.GATHER_SYSTEM_STATS;
                        end;
                        /
                        begin
                        DBMS_STATS.GATHER_FIXED_OBJECTS_STATS;
                        end;
                        /
                        begin
                        DBMS_STATS.GATHER_DICTIONARY_STATS(estimate_percent => 100, method_opt => 'FOR ALL COLUMNS SIZE AUTO', cascade => TRUE, options => 'GATHER' );
                        end;
                        /
                    
                
            
            Base de dados do sistema Senior
                
                    
                        begin
                        DBMS_STATS.GATHER_SCHEMA_STATS(OWNNAME => 'NOME_BASE', CASCADE => TRUE, method_opt => 'FOR ALL COLUMNS SIZE 1');
                        end;
                        /
                    
                    Para as tabelas com mais de 10 milhões de registros executar:
                    
                        begin
                        DBMS_STATS.GATHER_TABLE_STATS(OWNNAME => 'NOME_BASE', TABNAME => 'NOME_TABELA', partname => NULL, granularity => 'GLOBAL', CASCADE => TRUE, method_opt => 'FOR ALL INDEXED COLUMNS SIZE AUTO');
                        end;
                        /
                    
                
            
            
                Importante
                
                Para ambiente de missão crítica é necessário um planejamento para execução e validação da rotina. Se necessário, entre em contato com a IT Services Senior para avaliação.
            
            Estas recomendações são baseadas em boas práticas do banco de dados Oracle e avaliações em ambientes diversos. Acesse o site de suporte principal da Oracle para mais detalhes.
