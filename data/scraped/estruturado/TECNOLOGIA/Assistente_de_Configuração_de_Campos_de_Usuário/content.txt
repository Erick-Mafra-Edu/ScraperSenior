                Assistente de Configuração de Campos de Usuário
                O assistente de Configuração de Campos de Usuário permite que campos de usuários de qualquer tabela sejam adicionados em algum componente do sistema (como algumas telas e grades) desde que configurados os relacionamentos corretos.
                Utilizando o assistente de configuração
                Para que seja possível configurar estes campos de usuários utilizando o assistente, primeiramente é necessário ser possível configurar campos no componente desejado. Caso seja possível, a opção Configurar campos de usuário estará disponível no menu de contexto (popup) do componente. Ao utilizar esta opção, será aberta a tela Configurações de campo de Usuário:
                Tabelas Disponíveis
                    Tabelas que o usuário tem permissão de consulta (esta definição é feita através de permissão no CBDS e de definições fixas dentro do sistema), a partir destas tabelas serão adicionados campos de usuário;
                Campos Disponíveis
                    Ao selecionar a tabela desejada, neste painel serão apresentados os campos de usuário disponíveis para serem adicionados;
                Campos Selecionados
                    Lista de campos selecionados para serem mostrados na tela.
                Adicionando e removendo campos
                Ao incluir, os campos são movidos do painel Campos disponíveis para o painel Campos selecionados, para incluir campos, selecione os campos desejados e utilize o botão .
                Ao excluir, os campos são movidos do painel Campos selecionados para o painel Campos disponíveis, para excluir campos, selecione os campos desejados e utilize o botão .
                Após incluir e excluir os campos desejados, utilize o botão Avançar para seguir criar as ligações dos campos-chave da tabela do campo de usuário e os campos já existentes.
                Adicionando e Removendo Ligações
                Com base nos campos selecionados anteriormente, é possível adicionar e remover ligações nos campos.
                Campos selecionados
                    São os campos que foram selecionados para serem adicionados;
                Ligações Existentes
                    Ao selecionar um campo serão apresentados as ligações já adicionadas para este campo de usuário;
                Aplicar Alterações à
                    Aplica as alterações de acordo com a opção selecionada:
				Padrão: Aplica-se a todos que não tiverem configurações individuais;Individual: Apenas para o usuário corrente;Todos: Sobrescreve todas as opções para que todos tenham a mesma configuração.
                Ao selecionar um campo no painel Campos Selecionados e utilizar o botão Adicionar, será exibida a tela Efetuar ligações:
                Campos da Grid
                    Apresenta os campos pertencentes ao componente em que se deseja adicionar os campos de usuário;
                Chave - campos de usuário
                    Apresenta os campos-chave da tabela a qual pertence o campo de usuário a ser adicionado;
                Ligações adicionadas
                    Apresenta as ligações que já foram adicionadas.
                Para adicionar uma ligação, selecione um campo de cada lista (Campos da Grid e Chave - campos de usuário) e utilize o botão <>.
                Para excluir uma ligação, selecione a ligação desejada no painel Ligações existentes e utilize o botão Remover.
                Copiando e Colando Ligações
                Para copiar ligações, selecione o campo de onde serão copiadas as ligações e utilize o botão Copiar Ligações, em seguia selecione o campo para o qual serão aplicadas as ligações copiadas e utilize o botão Colar Ligações.
                
                    Observação
                    
                    Ao criar uma ligação, é essencial garantir que os campos informados sejam adequados, pois eles poderão ser utilizados na construção do SELECT responsável por buscar as informações e carregar os dados na tela, definindo as colunas-chave da consulta.
