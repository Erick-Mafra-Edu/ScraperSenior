                Erro (web): "Não foi especificada uma 
		 chave para a tabela 'nome_da_tabela'"
                Um dos motivos que levam à não especificação da chave em uma tabela, é a máscara de alguns campos que contém o caracter  
 "#", que não é suportado pelos Sistemas Web, ou seja, não é propriamente um erro do 
 SGI.
                O SGI Web envia uma String com os campos e as respectivas máscaras para 
 o conector, que por sua vez repassa para o Sistema Web que não suporta o caracter "#", tudo que está após o "#" é eliminado, incluindo as 
 informações dos campos-chave.
                Solução
                A solução é corrigir a máscara errada. Por 
 exemplo, a máscara #[EditCad]N(9,0)#, deve ser corrigida para 9[4] ou ZZZ9.
                A correção pode ser feita no SGI e após a alteração, é preciso recompilar o formulário. Porém é recomendado fazer correção no TBS, 
 para evitar que o mesmo erro aconteça novamente.
