                Junções
                Tipos de JOIN
                Importante
                    O nome da junção KEY JOIN foi alterado para KEYED JOIN. A junção FULL OUTER JOIN não existe mais.
                
                
                    CROSS JOIN (Produto Cartesiano);
                    INNER JOIN (Junção);
                    LEFT OUTER JOIN (Junção Externa a Direita);
                    RIGHT OUTER JOIN (Junção Externa a Esquerda);
                    NATURAL JOIN (Junção Natural);
                    KEYED JOIN (Junção por Chave);
                    Combinações de JOINS.
                
                Nota
                    Os bancos, suportados pela SQL Senior 2, que utilizam JOINS com uma sintaxe semelhante são Microsoft SQL Server 7.0 ou superior, Sybase AnyWHERE 5.0 ou superior. 
                
                CROSS JOIN (Produto Cartesiano)
                Retorna o produto cartesiano entre as tabelas.
                
                    Exemplo
                        
                            Fazendo o produto das duas tabelas, a seleção conterá todos os benefícios que todos os funcionários tem. 
                            
                                SELECT B.NOMFUN, C.NOMBEN FROM (funcionario B CROSS JOIN beneficio C)
                            
                            Equivalência do Exemplo:
                            
                                SELECT B.NOMFUN, C.NOMBEN FROM funcionario B, beneficio C
