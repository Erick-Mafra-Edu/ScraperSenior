            Visualizar Filtros
            Nesta tela são exibidos os filtros que estão aplicados nos campos e grades das telas do sistema. Esta tela é acessada através do menu de contexto (popup) > Mostrar filtros ou pelas teclas de atalho Ctrl + Shift + F em telas que permitem visualizar os filtros aplicados.
            
            
            Campos
            Tabela Ativa
                Neste quadro estão agrupados os campos que exibem os filtros da tabela de cadastro do campo ativo no sistema:
            
                Filter: campo com categoria interna de filtros para tabelas do sistema;
                SQLFilter: campo com categoria interna de filtros para tabelas do sistema;
                Range: campo com categoria interna de filtros para tabelas do sistema.
            
            Campo Ativo
                Neste painel estão agrupados os campos que exibem os filtros da tabela de consulta do campo ativo no sistema:
            
                Filter: campo com categoria interna de filtros para tabelas do sistema;
                SQLFilter: campo com categoria interna de filtros para tabelas do sistema;
                Range: campo com categoria interna de filtros para tabelas do sistema.
