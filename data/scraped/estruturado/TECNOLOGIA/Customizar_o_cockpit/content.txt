                Customizar o cockpit
                Apenas o           Administrador / Customizador terá a opção de personalizar as telas do Cockpit, adequando este à identidade visual da empresa através de temas. 
                O tema é um arquivo .css disponibilizado no servidor dentro do arquivo .EAR do Workflow. O arquivo pode ser encontrado no seguinte caminho:
                
                    <caminho de intalação do GlassFish>\<domínio do Workflow>\applications\j2ee-apps\Workflow\g6-wf-cockpit-theme_war\css\xtheme-senior-default.css
                
                
                    Exemplo:
                        
                            
                                c:\GlassFish\domains\domain1\applications\j2ee-apps\Workflow\g6-wf-cockpit-theme_war\css\xtheme-senior-default.css
                            
                        
                    
                
                Para a customização o conteúdo desta pasta deverá ser copiado para outra pasta e então editado, não é recomendado editar diretamente o tema padrão.
                A customização é opcional. Caso a opção seja não realizar a customização, será utilizado o tema padrão da Senior.
                Segue algumas alterações permitidas:
                
                    Tela de login
                    
                    Outras telas
                    
                    Editar tema
                    
                
                Tela de login
                
                Na tela de login poderão ser alterados:
                
                    Cor de fundo;
                    Logo da empresa:Tamanho da imagem;Alinhamento da imagem;URL da imagem.
                    Estilos dos componentes:Botões;Imagens;
                
                
                    Nota 
                    Todas as imagens configuradas no tema devem estar disponíveis dentro do .WAR do tema.
                
                Outras telas
                
                Nas outras telas do cockpit poderão ser alteradas:
                
                    Cor de fundo;
                    Estilos dos componentes:Botões;Imagens;Tabelas;Listas;Painéis.
                    Ícone padrão dos processos:
                        
                            É exibido quando  o modelo do processo não possui nenhum ícone configurado. Localiza-se  dentro do tema, no caminho: /images/process, seu nome é fech_bd e sua extensão .png.
                        
                    
                
                
                    
                
                
                    Nota 
                    Todas as imagens configuradas no tema devem estar disponíveis dentro do WAR do tema.
                
                Editar tema
                
                Para personalizar o tema:
                
                    Crie um novo projeto web no eclipse;
                    Copie o conteúdo da pasta <caminho de intalação do GlassFish>\<domínio do Workflow>\applications\j2ee-apps\Workflow\g6-wf-cockpit-theme_war\css;  e colar em outra pasta;
                    Realize as customizações no arquivo .css;Editar cor de fundo;Editar componentes (botões, tabelas, etc);Editar ícone padrão dos processos.
                    Exporte o projeto Web com um nome diferente do tema padrão;
                    Faça o deploy no servidor Java EE;
                    Aponte este na configuração do cockpit .
                
                
                    
                        
                    
                
                
                    Nota 
                    Para realizar a customização do tema, o servidor Java EE (GlassFish) não pode estar em execução.
                
                
                    Importante 
                    O projeto web criado para customização não poderá sobrescrever o tema default do Cockpit.
                
                Para aplicar as customizações no Cockpit não é necessário refazer o deploy do Workflow. Mas caso seja necessário refazer o deploy por algum outro motivo ou ainda atualizar a versão do Workflow, as customizações serão mantidas no Cockpit.
