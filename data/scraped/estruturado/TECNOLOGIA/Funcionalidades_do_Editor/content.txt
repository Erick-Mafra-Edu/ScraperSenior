            Funcionalidades do Editor
            É o aplicativo Editor de Modelos 
 que permite criar e editar modelos 
 para execução 
 no cliente. Esses modelos são criados por consultores especializados, 
 que terão acesso às informações e realidade do cliente.
            Pré-Condições
            A criação dos modelos de diagnósticos tem algumas pré-condições para 
 garantir a integridade e a segurança dos dados dos clientes Senior:
            
                Suportar somente funções de regra 
	 da tecnologia G5: Funções de regra criadas pelos sistemas não 
	 serão suportadas;
                Execução somente em ambiente Senior: 
	 O Editor de Modelos só é utilizado em ambiente Senior, para garantir 
	 a segurança do cliente, é exibida a seguinte mensagem caso o sistema 
	 não detecte o ambiente de rede da Senior: "Execução do sistema 
	 não permitida. Entre em contato com o serviço de suporte." e 
	 é finalizado;
                Utilização de banco de dados homologado: 
	 O sistema suporta conexão com todos as versões de banco de dados homologadas 
	 pela Senior. E  necessita de conexão com o banco de dados para 
	 compilação das regras. A conexão com o banco é feita através do arquivo 
	 CFG.  Alguns recursos de regra como os cursores, validam (se 
	 o comando SQL está correto e os objetos presentes no comando existem 
	 no banco de dados) os comandos SQL na compilação para verificar sua 
	 integridade.
            
            Execução do Modelo
            A execução do modelo e a geração 
 do arquivo de saída com os resultados pode ser feita totalmente pelo 
 cliente, pois exige apenas que o usuário selecione o modelo gerado pelo 
 consultor.
            O aplicativo permite exportar o resultado de uma execução nos formatos 
 binário e CSV, bem como gerar um Plano de Ação com as atividades e os 
 respectivos tempos necessários.
