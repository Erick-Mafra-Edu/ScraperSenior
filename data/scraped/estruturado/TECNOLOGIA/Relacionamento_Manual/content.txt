                Relacionamento Manual
                Esta tela mostra a(s) tabelas com seus campos e relacionamentos. Pode-se efetuar um relacionamento manual, incluir, excluir ou mover tabelas, relacionar os campos com campos de outras tabelas, ou relacionar com variáveis de sistema, ou ainda, com valores informados.
                Os botões desta tela, descritos abaixo, executam as seguintes operações:
                Abre uma tela que permite adicionar uma tabela a este relacionamento.
                Abre uma tela que permite excluir uma tabela do relacionamento
                Abre um tela que permite movimentar as tabelas envolvidas no relacionamento.
                Permite efetuar um relacionamento entre campos de tabelas diferentes. Para isso deve-se clicar sobre este botão, e em seguida, clicar sobre um campo de tabela e arrastar para um campo de outra tabela. Para que não ocorra erros, o tipo de dado destes campos devem ser iguais. Esta ação cria um relacionamento entre as duas tabelas através destes dois campos.
                Permite relacionar um campo de uma tabela com o valor de uma variável de sistema. Para fazer este relacionamento deve-se clicar sobre este botão, e dar um duplo clique sobre o campo da tabela que se deseja fazer o relacionamento, surgindo uma lista de variáveis de sistema que poderão ser utilizadas para este relacionamento.
                Permite relacionar um campo de uma tabela com o valor fixo. Para fazer este relacionamento deve-se clicar sobre este botão, e dar um duplo clique sobre o campo da tabela que se deseja fazer o relacionamento, surgindo uma tela solicitando o valor a ser utilizado para este relacionamento.
