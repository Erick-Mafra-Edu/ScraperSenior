            Editor de Regras - Menu Editar
            
                
                
                
                
                    
                        Comando
                        Descrição
                        Tecla de atalho
                    
                    
                        
                             Desfazer
                        Desfaz a última ação realizada.
                        Ctrl + Z
                    
                    
                        
                             Refazer
                        Refaz a última ação desfeita.
                        Shift + Ctrl+ Z
                    
                    
                        
                             Recortar
                        Move o texto selecionado no editor para a área de transferência do Windows.
                        Ctrl + X
                    
                    
                        
                             Copiar
                        Copia o texto selecionado no editor para a área de transferência do Windows.
                        Ctrl + C
                    
                    
                        
                             Colar
                        Insere no editor logo após a posição do cursor o texto armazenado na área de transferência do Windows.
                        Ctrl + V
                    
                    
                           Selecionar Tudo
                        Seleciona todo o texto contido no editor de regras.
                        Ctrl + T
                    
                    
                           Modo de Seleção
                        Permite definir diferentes modos de seleção para o texto.
                         
                    
                    
                           Macros
                        Permite gravar ações para reproduzi-las posteriormente de forma automatizada.
                         
                    
                    
                           Indentar
                        Permite deslocar blocos inteiros de texto para direita ou para esquerda.
                         
                    
                    
                        
                             Localizar
                        Permite localizar um texto em específico no texto do editor.
                        Ctrl + L
                    
                    
                        
                             Localizar Próxima
                        Repete a última busca após a última ocorrência encontrada.
                        F3
                        
                    
                    
                           Substituir
                        Substitui um texto por outro.
                        Ctrl -U
                    
                    
                           Ir Para Linha
                        Posiciona o cursor na linha especificada.
                        Alt + G
                    
                    
                           Retornar Última Versão Salva
                        Volta o texto da regra exatamente como salvo pela última vez.
                        Ctrl + M
