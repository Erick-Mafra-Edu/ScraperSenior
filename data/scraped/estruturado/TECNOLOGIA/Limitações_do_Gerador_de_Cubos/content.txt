            Limitações do Gerador de Cubos
            Barra de Rolagem Horizontal              
                
                    A Barra de Rolagem Horizontal do Gerador de Cubos realiza a rolagem das colunas
        individualmente, mas se aplica somente aos resumos. Caso o primeiro resumo não esteja
        visível na tela a barra de rolagem não surtirá efeito algum, pois ele não descola
        as quebras para tornar visível os resumos e sim vai rolando os resumos e mantendo
        as colunas de quebra imóveis.
                
            
            Envio de E-mail
                
                
                    O envio de e-mail do saga utiliza somente MAPI.
                
            
            Redimensionamento da Coluna Sequência
                
                
                          Na extração do cubo, quando habilitado a opção de Numerar Linhas, esta coluna na
        visão não é possível redimensioná-la.
                
            
            Quantidade de Linhas x Colunas
                
                
                    Existe uma limitação entre o tamanho de linhas x colunas x dados. Esta limitação
        é variável, e não é possível definir um valor exato de quantidade máxima linhas
        ou colunas de uma visão. O que ocorre é que com uma massa de dados muito grande
        e diversificada, sem filtro, o Gerador de Cubos não consegue indexar todos os valores
        em memória, o que causa perda de dados em alguns casos. Isto pode ser observado
        em células que possuem o valor #VALOR!.
    
                
            
            Erro #VALOR!      Este valor aparece quando o Cubo não consegue recuperar o valor da célula.     
                
                    Como resolver?
                               Pode-se aplicar uma abrangência menor na tela de entrada ou alterar as linhas/colunas
        exibidas na dimensão.
    
                
            
            "A massa de dados gerada pela visão é muito grande e algumas
            linhas/colunas não foram carregadas corretamente. Verifique o filtro aplicado na
            visão."      Esta situação ocorre quando o Cubo não consegue indexar todos os valores da Visão.
        Mesmo assim é possível visualizar a Visão, porém não existe garantia de que todos
        os dados estão consistentes.       
                
                    Como resolver?
                              Pode-se aplicar uma abrangência menor na tela de entrada ou alterar as linhas/colunas
        exibidas na dimensão.

                
            
            Ao executar o gerador de Cubo para grande quantidade de informações, pode ocorrer a mensagem: "O cubo não pode ser visualizado pois seu tamanho ultrapassa os limites de recursos disponíveis".
                
                    Como resolver?
                        Este tipo de mensagem é apresentada quando o limite de objetos utilizados do sistema operacional Windows for alcançado. Não é um limite da aplicação Senior, trata-se de uma limitação de aplicações 32 bits no próprio sistema operacional.
                    Como a área de memória reservada para alocação destes objetos é estática e limitada, dependendo da quantidade de aplicações abertas no computador, ocorre erro de falta de recursos, quando a aplicação tenta alocar um novo objeto e a área de memória em questão já está cheia.
                    A alternativa de contorno é gerar este cubo com menos dados, e juntá-los manualmente depois de exportados.
