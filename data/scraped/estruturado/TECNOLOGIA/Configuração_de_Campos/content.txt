                Configuração de tela
                Este recurso permite adicionar e retirar campos de usuário e de sistema da tela e alterar a ordem dos campos na tela:
                Campos disponíveis
                    Lista de campos disponíveis para serem mostrados na tela. Os campos desta lista não são mostrados na tela, mas podem ser configurados para ficarem visíveis;
				Ordenação: Método de ordenação dos campos disponíveis:
						Layout: Ordena os campos disponíveis seguindo a ordem definida no layout da tabela (ordem definida no TBS do sistema);Alfabética: Ordena os campos disponíveis seguindo a ordem alfabética.
                Campos Selecionados
                    Lista de campos selecionados para serem mostrados na tela. Os campos desta lista são mostrados na tela na ordem que aparecem na lista.
                
                    Ordenação: Método de ordenação dos campos disponíveis:
					Ordenar pelo Layout: Ordena os campos selecionados seguindo a ordem definida no layout da tabela (ordem definida no TBS do sistema). Esta operação afeta a ordem dos campos na tela;Restaurar configuração original: Restaura a configuração original da tela do sistema.
                
                Salvar como
                    Define para quais usuários a configuração será salva:
				
                
                    
                        Padrão: Salva a configuração como padrão, aplicando as configurações para todos os usuários que não tem configuração individual. Esta opção não afeta configurações individuais de outros usuários;
                        Todos: Salva a configuração para todos os usuários: Esta operação sobrescreve as configurações de todos os usuários;
                        Individual: Salva a configuração somente para o usuário logado no sistema. Esta opção não afeta configurações individuais de outros usuários.
                    
                
                Adicionando e removendo campos da tela
                Ao incluir campos na tela, os campos são movidos do painel Campos disponíveis para o painel Campos selecionados, a inclusão de campos na tela pode ser feita de duas maneiras:
                
                    Adicionando todos os campos: Para adicionar todos os campos disponíveis, utilize o botão ;
                    Adicionando campos individuais: Para adicionar somente alguns dos campos disponíveis, selecione os campos desejados e utilize o botão .
                
                
                    Nota
                    
                    O sistema não oferece a possibilidade de inserir campos do tipo imagem nas telas,  eles não são exibidos no painel Campos disponíveis.
                
                Ao excluir campos na tela, os campos são movidos do painel Campos selecionados para o painel Campos disponíveis, a exclusão de campos na tela pode ser feita de duas maneiras:
                
                    Removendo todos os campos: Para remover todos os campos selecionados (não obrigatórios), utilize o botão ;
                    Removendo campos individuais: Para remover somente alguns campos selecionados (não obrigatórios), selecione os campos desejados e utilize o botão . Quando algum campo selecionado for obrigatório, será exibida uma mensagem informando que não foi possível realizar a operação nos campos obrigatórios.
                
                Alterando a ordem dos campos da tela
                Para mover um campo para uma posição anterior, selecione o campo no painel Campos selecionados e utilize o botão . Caso o campo selecionado seja dependente do campo superior, por exemplo, o campo Bairro que é dependente do campo Cidade, não será possível movê-lo e será exibida uma mensagem informando que não é possível realizar este posicionamento (Bairro antes da Cidade).
                Para mover um campo para uma posição posterior, selecione o campo no painel Campos selecionados e utilize o botão . Caso o campo posterior ao selecionado seja dependente do campo selecionado, por exemplo, o campo Cidade em relação ao campo Bairro (dependente de Cidade), não será possível movê-lo e será exibida uma mensagem informando que não é possível realizar este posicionamento (Cidade depois do Bairro).
                Nota
                    Em algumas telas não é possível alterar a ordem dos campos de sistema, apenas dos campos de usuário. Neste caso ao tentar alterar a ordem de algum campo de sistema, será exibida uma mensagem informando que não é possível alterar a ordem do campo.
                
                Configuração de campos de usuário
                Algumas telas do sistema permitem uma maior flexibilização na exibição de campos de usuário, onde podem ser exibidos campos de usuário de todas as tabelas do sistema, desde que o usuário tenha permissão para acessar a tabela e que não exista uma restrição específica do sistema na exibição desta tabela.
                Nestas telas, antes dos campos serem mostrados, eles devem ser configurados através do Assistente de Configuração de Campos de Usuário. Nas telas que necessitam desta configuração, o assistente pode ser acessado através da opção Configurar Campos de Usuário, no menu de contexto da tela (menu popup).
