            Editor de Regras - Barra de Ferramentas|Marcadores
            Os marcadores permitem marcar linhas do editor, para posteriormente, acessar estas linhas de forma rápida e prática.Para inserir um novo marcador, clique na linha onde deseja inseri-lo, clique na seta do botão  e em Marcar,escolha um dos marcadores disponíveis. É possível inserir até 10 marcadores cujos números vão de 0 até 9. Ao clicar sobre um deles, este será marcado e na margem esquerda numerada do editor, aparecerá um quadrado com o número correspondente no centro. 
            Para remover um marcador, no menu Marcar na barra de ferramentas e clique sobre o item marcado anteriormente.
            Também é possível inserir ou remover marcadores através das teclas de atalho que estão no próprio menu Shift + Ctrl + número de 0 até 9, ou ainda clicando com o botão direito do mouse no editor escolhendo o menu Marcadores > Marcar > Escolha o marcador desejado.
            Após inserir marcadores é possível utilizá-los para movimentar-se rapidamente pelas linhas marcadas. Para ir à uma linha marcada, clique na seta do botão  e em Ir Para, apenas os marcadores já inseridos estarão habilitados; clique sobre um deles. Pode-se também utilizar as teclas de atalho que o próprio menu indicar (Ctrl + Números de 0 até 9). Ou ainda clicando com o botão direito do mouse na opção Marcadores > Ir Para > Escolha o marcador desejado.
            Após adicionar marcadores também é possível mover-se por eles através dos botões:
            
                
                     = vai para o próximo marcador, se já estiver posicionado no último marcador, vai para o primeiro;
                
                     = vai para o marcador anterior, se já estiver no primeiro, vai para o último marcador.
