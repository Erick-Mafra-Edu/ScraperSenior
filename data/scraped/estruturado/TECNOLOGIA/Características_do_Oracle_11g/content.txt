            Características do Oracle 11g
            Lixeira
            Este recurso não é utilizado pelas nossas rotinas. Quando uma tabela é excluída, é utilizada a opção "PURGE" para que a tabela seja excluída definitivamente. Na engenharia reversa, as tabelas que estão na lixeira são ignoradas.
            Comandos:
            
                Excluir uma tabela no oracle 11g para não utilizar o recurso de lixeira: drop table NOMETABELA purge;
                Para verificar se tem alguma informação na lixeira: select * from recyclebin
                Para limpar a lixeira: purge recyclebin
                Para desabilitar a lixeira: alter system set "recyclebin"=false scope=both;
