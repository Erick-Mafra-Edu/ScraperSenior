                Assistente para adicionar campos de usuário em Grids
                Em algumas grades é necessário utilizar um assistente para adicionar campos de usuário,  efetuando a ligação entre a chave da tabela ao qual o campo de usuário pertence e os campos já existentes na grade.
                Utilizando o assistente de configuração
                Para que seja possível adicionar campos de usuário em uma grade utilizando o assistente, é preciso que seja possível configurar os campos na grade. Para verificar, identifique se há um botão C no canto superior esquerdo da grade. Havendo este botão, a opção Configurar campos de usuário estará presente no menu de contexto (popup), esta opção abre o Assistente de Configuração de campos de usuário.
                Considerações
                
                    Os campos de usuários serão adicionados na grade somente após a reabertura da tela;
                    Quando o campo de usuário for do tipo enumeração, o campo adicionado terá uma lista para a seleção de valores e automaticamente será adicionada uma coluna com a descrição da enumeração.
