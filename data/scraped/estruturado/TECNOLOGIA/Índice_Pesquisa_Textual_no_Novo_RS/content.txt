            Índice Pesquisa Textual no Novo RS
            Para utilizar a tela Pesquisar Candidatos do Novo RS é necessário criar 
 um índice especial, de acordo com o banco de dados utilizado:
            
                
                    Banco
                    Índice
                
                
                    Oracle
                    CREATE INDEX FTIND_R122CCO ON R122CCO(CurCpl)		INDEXTYPE IS CTXSYS.CONTEXT PARAMETERS('SYNC(ON COMMIT)');
                
                
                    Oracle 10g ou superior
                    CREATE INDEX FTIND_R122CCO ON R122CCO(CurCpl)
 INDEXTYPE IS CTXSYS.CONTEXT PARAMETERS('SYNC(ON COMMIT)');
                
                
                    SQL Server
                    CREATE UNIQUE INDEX FTIND_R122CCO ON R122CCO(SeqCCO);		CREATE FULLTEXT CATALOG ft AS DEFAULT;		CREATE FULLTEXT INDEX ON R122CCO(CurCpl) KEY INDEX FTIND_R122CCO;
                
            
            Para criação do índice é necessário habilitar o recurso de pesquisa 
 textual no banco de dados.
