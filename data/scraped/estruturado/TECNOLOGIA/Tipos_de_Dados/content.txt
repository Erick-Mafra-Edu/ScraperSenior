                Tipos de Dados
                
                    
                    
                    
                    
                        Tipo
                        Explicação
                        Exemplos
                    
                    
                        number
                
                        
					Guarda números dentro de precisão e escala especificados. A precisão máxima é 15 dígitos. A escala é um número não-negativo que vai de zero a número de dígitos-1.
				
                        number(5,2)
                    				Precisão 5, escala 2. Faixa de valores: de -999,99 a +999,99.number(4)	Precisão 4, escala 0. Faixa de valores: de -9999 a +9999.
                
                    
                    
                        string
                
                        Guarda dados do tipo texto, dentro de um tamanho especificado. O tamanho máximo é 254 caracteres.
                        string(20)
                    Pode conter de 0 a 20 caracteres.
                    
                    
                        text
                
                        Guarda dados do tipo texto, podendo conter quebras de linha. O tamanho máximo é definido pelo banco nativo. Possui restrições quanto ao uso em expressões.
                        text(1000)
                    				Pode conter de 0 a 1000 caracteres.
                
                    
                    
                        date
                
                        Guarda uma data, dentro de um contexto e precisão especificados. Os componentes de uma data podem ser ano, mês, dia, hora, minuto, segundo, e milissegundo. O contexto é o primeiro componente que a data usa. A precisão é o último componente.
                        date(year to day)Guarda uma data que contém os componentes ano, mês e dia.date				Guarda uma data que contém os componentes ano, mês e dia.date(month to second)				Guarda uma data que contém os componentes mês, dia, hora, minuto e segundo.date(hour to msecond)				Guarda uma data que contém os componentes hora, minuto, segundo e milissegundo.
				
                    
                    
                        blob
                
                        Guarda um conjunto de bytes. Pode ser usado para guardar fotos ou qualquer outro dado no formato binário. O tamanho máximo é definido pelo banco nativo.
