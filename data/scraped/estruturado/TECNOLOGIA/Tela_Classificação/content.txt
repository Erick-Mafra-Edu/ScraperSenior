                Classificação
                Permite determinar a ordem de impressão e quebra do modelo, e  fazer a conexão entre as seções subtítulo e subtotal com a seção detalhe. Se a seção subtítulo ou subtotal não forem relacionadas com a seção detalhe, as mesmas não serão impressas. As opções da tela são:
                Campos
                    Visualiza todos os campos tipo cadastro, fórmula e especiais que estão na Seção Detalhe.
                Campos Ordenados
                    Ordem em que os campos do tipo cadastro serão listados no relatório. A ordem é de cima para baixo. Para incluir um campo para ordenação, deve-se clicar em um campo de tabela, ou em uma definição de campo ordem, e clicar em OK.
                Definir Campo Ordem
                    Esta tela permite visualizar os campos ordem definidos.
                Quebras
                    Permite selecionar uma quebra para o relatório, setando para a mesma, um subtítulo ou um subtotal, para isso, deve-se selecionar um dos campos na tela de campos ordenados, e selecionar um subtítulo ou subtotal para o mesmo. Estes campos somente ficarão habilitados para o Detalhe Mestre do modelo.
                Ordem
                    Define a ordem em que o campo selecionado na tela campos ordenados assumirá.
                Subtítulo Geral
                    Permite listar um subtítulo para a tabela filha. Para selecionar, deve-se selecionar um dos campos na tela de campos ordenados, e selecionar um subtítulo desejado.
                Subtotal Geral
                    Permite listar um subtotal para a tabela filha. Para selecionar, deve-se selecionar um dos campos na tela de campos ordenados, e selecionar um subtotal desejado.
                Os botões desta tela, descritos abaixo, executam as seguintes operações:
                Inserir
                    Insere uma definição em campos ordem.
                Alterar
                    Permite alterar uma definição de campos ordem, para isso deve-se selecionar um dos campos ordem e clicar sobre este botão.
                Eliminar
                    Exclui um dos campos ordem definidos.
