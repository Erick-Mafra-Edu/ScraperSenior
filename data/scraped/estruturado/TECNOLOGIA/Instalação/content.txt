            Checklist de Instalação
            Para que a instalação do sistema ETL X on-premises (ETL X + MicroStrategy)   ocorra corretamente, sugerimos verificar alguns itens antes de iniciar este processo. 
            Para facilitar, imprima esse documento em formato PDF, no leiaute "Paisagem", a partir do ícone de impressão visível no início desta página.
            
                Importante
                
                Antes de iniciar este processo, verifique se os requisitos de infraestrutura para utilização do ETL X estão sendo atendidos.
            
            Instruções
            Descrição dos itens
            
            
                Requisito: itens necessários para ocorrer instalação do ETL X;
                Necessidade: requisito obrigatório ou opcional para realizar instalação;
                Evidência: evidências que comprovam que o requisito está sendo atendido para que a instalação ocorra corretamente;
                Responsável: preenchimento com o nome do responsável por verificar se o requisito está sendo atendido;
                Data: preenchimento com a data em que o requisito foi avaliado.
