            Acesso Rápido
            O recurso Acesso Rápido facilita a abertura das telas do sistema, não sendo mais necessário navegar entre os menus. Através de uma tecla de atalho, informe o nome da tela desejada e confirme (botão Ok) para abrir a tela.
            
                
            
            A busca é incremental, ao digitar as primeiras letras o recurso já filtra automaticamente o nome e o caminho das telas que começam com as letras digitadas. É possível também abrir uma lista que exibe o nome e o caminho das telas do sistema.
            Para utilizar este recurso, crie um menu com o nome FASTACCESSFORM e atribua-o uma tecla de atalho.
            Notas
                A tecla de atalho que ativa este recurso pode ser diferente em cada sistema da Senior. Como sugestão, escolha F11 como atalho do menu, assim todos os sistemas terão o mesmo atalho para a tela.
                Este recurso, quando acessado pelo BrowserAccess ou WindowsAccess, possui uma limitação: não é possível digitar valores para pesquisa, somente selecionar uma tela através da lista.
