                Instalação de serviços no Windows
                Atualmente é possível ignorar algumas situações que apresentam problema durante a conversão, atualização ou instalação de sistemas Senior, com isso, é necessário instalar alguns dos serviços dos manualmente:
                Como instalar o serviço do SeniorMiddleware.exe
                Na tela de execução do Windows (teclas Windows + R), execute o comando "<DIRETÓRIO DA INSTALAÇÃO>\SeniorMiddleware.exe id:<AMBIENTE> instid:<ID da instalação> -svc -amgr /INSTALL", por exemplo:
                
                    
                
                Sendo que:
                
                    - svc: Define como Gerenciador de Middleware;
                    - amgr: Define como Gerenciador de Aplicativos;
                    instid: ID do serviço de instalação. Esse ID pode ser consultado na Central de configurações, tela Informações da instalação.
                
                Atenção 
                    Ao configurar o logon no serviço, é importante observar as configurações de idiomas deste usuário, pois se o mesmo possuir diferença entre os padrões que são utilizados pelos outros usuários (exemplo: Formato de Data, Formato de Moeda), poderão ocorrer erros na execução de aplicação emuladas pelo Middleware.
