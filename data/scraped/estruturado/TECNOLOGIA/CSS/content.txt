                CSS
                O SGI Web foi modificado para suportar o recurso CSS (Cascading Style Sheets), que permite a personalização de todos os elementos gráficos no SGI Web (fontes, cores, botões, imagens, etc).
                Todas as configurações dos estilos estão no arquivo "SGI.CSS", para personalizar as telas do SGI é preciso alterar este arquivo.
                Para demonstrar o recurso, foi criado o arquivo "SGI.CSS_EXEMPLO_MODIFICADO", que é uma personalização das telas do SGI. 
                Para testar é preciso renomear o arquivo "SGI.CSS_EXEMPLO_MODIFICADO" para "SGI.CSS", sobrescrevendo o arquivo antigo, assim todos os estilos serão modificados. 
                Para retornar à configuração padrão, deve-se utilizar a cópia do arquivo com a configuração original "SGI.CSS_COPIA_ORIGINAL" sobrescrevendo  novamente o "SGI.CSS"  (este que foi renomeado previamente a partir do "SGI.CSS_EXEMPLO_MODIFICADO").
                Nota
                    O calendário e a tela de pesquisa não são de responsabilidade do SGI, ele apenas chama estas telas. A tela de pesquisa vai ser modificada pela equipe WEB da Tecnologia para também suportar CSS. Sendo assim, ao personalizar o SGI alterando o arquivo "SGI.CSS", é interessante também personalizar a tela de pesquisa alterando um outro arquivo CSS. Quanto ao calendário, ele já utiliza CSS, e a personalização do mesmo deve ser feita nos arquivos "CALENDAR.JS" para calendários com máscara "DD/MM/YYYY" e "CALENDAR_COMPET.JS" para calendários com máscara "MM/YYYY". Ao editar estes arquivos, deve-se alterar a configuração na seção "CALENDAR COLORS", no início do arquivo.
