            Tela de seleção de arquivos de regras
            Esta tela é utilizada pelo editor de regras para que o usuário informe o arquivo que deverá ser aberto ou criado.
            Categoria
                Campo utilizado para definir a categoria da regra. Este campo estará habilitado apenas para sistemas que definem categorias de regras.
            Código
                Campo utilizado para informar o código do arquivo que deve ser aberto/criado.  Tanto na abertura quanto na criação de um arquivo poderá ser utilizado o botão deste campo para pesquisar os códigos existentes.
            Descrição
                É a descrição da regra. Quando está abrindo um arquivo, este campo apenas apresentará a descrição do arquivo selecionado. Quando está criando um arquivo novo, pode-se informar uma descrição nesse campo.
