            Funções para FTP
            
                
                    
                        Nome
                        Descrição
                    
                    
                        FtpApagarArquivo
                        
                        Permite apagar um arquivo do servidor FTP.
                    
                    
                        FtpApagarDiretorio
                        
                        Permite apagar um diretório do servidor FTP.
                    
                    
                        FtpConexao
                        
                        
                            Configura a conexão com o servidor FTP para ser utilizada nas demais funções.
                        
                    
                    
                        FtpCriarDiretorio
                        
                        
                            Permite criar um novo diretório no servidor FTP.
                        
                    
                    
                        FtpDownload
                        
                        
                            Copia um arquivo do servidor FTP para um diretório local.
                        
                    
                    
                        FtpListaItem
                        
                        
                            Obtém o nome e o tipo de um item da lista.
                        
                    
                    
                        FtpListaItemDetalhes
                        
                        
                            Obtém informações adicionais sobre um item da lista, como tamanho, data de modificação e hora de modificação.
                        
                    
                    
                        FtpListaItemPermissoes
                        
                        
                            Obtém informações sobre as permissões do item no servidor FTP.
                        
                    
                    
                        FtpListaQuantidade
                        
                        
                            Retorna a quantidade de itens da lista.
                        
                    
                    
                        FtpListar
                        
                        Permite listar os arquivos e diretórios do servidor FTP.
                    
                    
                        FtpModoConexao
                        
                        Define se a conexão ao servidor FTP é feita no modo passivo ou ativo. 
                    
                    
                        FtpModoErro
                        
                        Quando alguma ação no FTP falhar, configura a forma como o erro deve ser retornado para a regra.
                    
                    
                        FtpModoTransferencia
                        
                        Define o modo de transferência utilizado para transferir os dados entre o servidor e o cliente. 
                    
                    
                        FtpProxy
                        
                        Configura os parâmetros de proxy para a conexão com servidor FTP. 
                    
                    
                        FtpRenomear
                        
                        Permite renomear um diretório ou um arquivo do servidor FTP.
                    
                    
                        FtpSetaSFTP
                        
                        Permite o acesso a servidores SFTP através da LSP.
                    
                    
                        FtpUltimoErro
                        
                        Retorna a mensagem de erro da última operação executada no FTP, caso tenha falhado.
                    
                    
                        FtpUpload
                        
                        
                            Copia um arquivo local para um diretório do servidor FTP.
