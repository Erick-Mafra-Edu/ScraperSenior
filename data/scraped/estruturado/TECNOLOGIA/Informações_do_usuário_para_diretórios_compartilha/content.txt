            Informações do usuário para diretórios compartilhados
            Esta tela só é visualizada por usuários administradores. Através dela é possível verificar:
            
                Se o recurso de Usuário para diretórios compartilhados está ou não ativo.
                Quais são os diretórios compartilhados e também sua descrição.
            
            
                Dicas
                
                
                    Caso queira, o administrador de rede pode efetuar auditoria dos diretórios compartilhados normalmente como faria para qualquer outro diretório compartilhado na rede.
                    
                        Para comodidade, é possível copiar o texto dos diretórios selecionados para a área de transferência do sistema operacional, através das teclas de atalho Ctrl + C ou ainda clicando com o botão direito do mouse sobre a opção desejado e selecionando Copiar Ctrl + C.
