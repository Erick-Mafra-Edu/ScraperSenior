                Tabela ASCII
                O Código Padrão Americano para Intercâmbio de Informação (American Standard Code for Information Interchange), foi desenvolvido pelo ANSI (American National Standards Institute), e é o formato mais comum para arquivos texto em computadores. Em um arquivo de ASCII, cada caracter é representado pelos 8 bits (equivalente a um byte), sendo que o oitavo bit (da direita para a esquerda) serve como um bit de paridade, utilizado para detectar erros. São definidos 128 caracteres possíveis.
                UNIX e sistemas operacionais baseados em DOS (com exceção de Windows NT) usam o ASCII para arquivos de texto. Windows NT usa um código mais novo chamado Unicode. O Sistema da IBM de 390 servidores usam um 8-bit código proprietário chamado EBCDIC. Programas de conversão permitem sistemas operacionais diferentes para mudar um arquivo de um código para outro.
