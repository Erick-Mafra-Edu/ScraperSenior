            Informações Avançadas
            Contêm informações detalhadas sobre memória, DLLs carregadas, versões e dados sobre a máquina onde o sistema está sendo utilizado. A tela pode ser acessada pressionando Shift + F3 em qualquer tela ativa do sistema ou em Ajuda > Sobre.
            O botão Salvar informações grava um relatório completo da tela em formato texto.
            Características do Sistema
            Esta guia traz informações sobre versões de banco de dados, usuário e memória:
            Tamanho do sistema
                Tamanho do executável do sistema.
            Data e hora do sistema
                Data e hora do executável do sistema.
            Data e hora do servidor
                Data e hora extraídos do servidor de Banco de Dados.
            Usuário do sistema
                Usuário efetivamente logado no sistema.
            Código do Cliente
                Código que representa o cliente na Senior.
            Arquivo de CFG
                Arquivo de configuração utilizado para acessar o sistema.
            Banco de Dados
                Versão do cliente do Banco de Dados.
            Versão do Banco no Servidor
                Versão do Banco de Dados instalado no Servidor.
            Diferenciar maiúsc./minúsc.
                É verificado se o Banco de Dados no servidor está configurado para diferenciar caracteres maiúsculo e minúsculo.
            Engine de Acesso
                Forma de acesso utilizado.
            Usuário do Banco de Dados
                Nome do usuário do Banco de Dados.
            ID do Usuário do BD
                ID do usuário do Banco de Dados.
            Versão do TBS de Sistema
                Versão da base de dados segundo o sistema.
            Versão do TBS de Rotinas
                Versão da base de dados segundo os recursos da Tecnologia.
            Data de Geração do TBS
                Data correspondente a geração do arquivo de TBS.
            Tipo de TBS
                Indica o valor referente ao tipo para arquivo TBS.
            Mem. Alocada Virtual
                Contém o tamanho da memória virtual que está efetivamente alocada pela aplicação. O valor deste campo é obtido através da função GlobalMemoryStatus do Windows, calculando-se a diferença entre o tamanho total da memória virtual para o processo e o tamanho de memória virtual não utilizado. A Memória Virtual Alocada considera memória comitada e reservada e o campo Tamanho da VM do Task Manager considera apenas as páginas comitadas.
            Versão de Sistema na Base
                Informa a versão do sistema que está contida na base de acordo com a tabela R999VER.
            Versão de Rotinas na Base
                Informa a versão das rotinas de tecnologia conforme especificado na tabela R999VER.
            PID da aplicação
                Informa o número de identificação do processo em execução no Windows.
            Mem. Alocada Virtual
                Informa o tamanho da memória alocada virtual.
            Mem. Alocada Real
                Contém o tamanho total de todos os blocos de memória utilizados pela aplicação. Informa a quantidade de memória utilizada pela aplicação corrente. O valor deste campo é obtido através da função AllocMemSize do Delphi.
            Blocos Alocados
                Quantidade de blocos de memória alocados pelo sistema.
            Memória Livre
                Contém a quantidade de memória virtual livre dentro do espaço total de memória virtual para a aplicação. O valor deste campo é obtido através da função GlobalMemoryStatus do Windows.
            Memória Compartilhada Alocada
                Quantidade de memória alocada que pode ser compartilhada entre as instâncias Java do módulo em execução.
            Memória Compartilhada Utilizada
                Parte da memória compartilhada alocada que está sendo utilizada.
            Ordenação do Banco
                Método de ordenação configurado no banco de dados. 
            Qtd. Transações Abertas
                Quantidade de transações ainda abertas com o servidor.
            Janela Ativa no Momento
                Janela ativa no momento em que a tela de Informações Avançadas foi aberta.
            Idioma
                Informa o idioma que está atuando.
            Versão Idioma de Sistema
                Mostra a versão do Idioma de Sistema. A versão está contida no arquivo SML do Sistema.
            Versão Idioma de Tecnologia
                Mostra a versão do Idioma da Tecnologia. A versão está no arquivo SML da Tecnologia.
            Arquivos de Mensagens
                Informa o caminho completo e a versão de todos os arquivos de mensagem. Caso não tenha os arquivo de mensagem (extensão SML) no diretório de idiomas, a versão irá mostrar como Não Disponível.
            Arquivos e Versões
            Esta guia é dividida em quatro seções que trazem informações sobre todas as DLLs e BPLs necessárias para a execução do sistema juntamente com a versão de cada arquivo:
            DLLs
                Traz todas as DLLs necessárias para a execução do sistema.
            BPLs carregadas
                Exibe todas as BPLs da Tecnologia utilizadas pela aplicação.
            BPLs ainda não carregadas
                Exibe as BPLs que não foram carregadas pela aplicação mas que podem ser carregadas em algum determinado momento.
            Outros arquivos
                Mostra os arquivos que foram necessários para a aplicação, mas não são nem DLLs, nem BPLs.
            Computador
            Esta guia exibe um relatório com informação sobre o hardware da máquina onde está rodando a aplicação. São exibidos os dados do Processador bem como a quantidade de Processadores, Sistema Operacional, Quantidade de Memória e Capacidade do disco rígido.
