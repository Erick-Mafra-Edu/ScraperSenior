                Funcionalidades
                O SQL Senior 2 possui duas formas de tratar datas: como constantes, ou como strings convertidas em tempo de execução.
                DAY
                A primeira forma (constantes) usa a palavra reservada DAY.
                
                    Exemplo
                        
                            Comando para listar todos os funcionários que nasceram depois de 3 de dezembro de 1977.
                            
                                SELECT * FROM R034FUN WHERE DATAS > DAY(1977,12,3)
                            
                        
                    
                
                STRTODATE
                A segunda forma (conversão de strings em tempo de execução), usa a função STRTODATE.
                
                    Exemplo
                        
                            Comando para listar todos os funcionários que nasceram depois de 3 de dezembro de 1977.
                            
                                
SELECT * FROM R034FUN WHERE DATAS >STRTODATE('1977/12/3')
                            
                        
                    
                
                SQL Senior 2 x Oracle
                Comando em SQL Senior 2.
                
                    
SELECT * FROM R034FUN WHERE DATAS >STRTODATE('1990/10/10')
                
                Comando equivalente no Oracle.
                
                    SELECT /*+ FIRST_ROWS */ DATADM FROM R034FUN R034FUN WHERE DATADM >
EXTEND(TO_DATE('1990/10/10', "%Y/%m/%d"), YEAR TO DAY)
