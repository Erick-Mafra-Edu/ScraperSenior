            Funções de Manipulação da Proprietária
            Funções que servem para consultar dados da proprietária, estas funções só estarão disponíveis para os sistemas que as habilitarem.
            Os sistemas são identificados nestas funções através dos códigos:
            >
                CódigoSistema1Gestão de Pessoas | HCM2Gestão Empresarial | ERP3Ronda Senior5Regente6BI7Workflow8Mobilidade
             
            
                
                    
                        Nome
                        Descrição 
                    
                    
                        Prop_ContemModulo
                        
                        Retorna se a proprietária ativa contém o código do módulo informado para um sistema específico.
                    
                    
                        Prop_ContemSistema
                        
                        Retorna se a proprietária ativa contém o código de sistema informado.
                    
                    
                        Prop_ModuloAtivo
                        
                        Retorna se um determinado módulo está ativo ou não pelo seu índice.
                    
                    
                        Prop_ModuloWeb
                        
                        Verifica se é um módulo é WEB.
                    
                    
                        Prop_ObterCodigoCliente
                        
                        Retorna o código do cliente da proprietária.
                    
                    
                        Prop_ObterDataValidade
                        
                        Retorna se um determinado módulo está ativo ou não pelo seu índice
                    
                    
                        Prop_ObterDataValidadeModulo
                        
                        Data de validade do módulo.
                    
                    
                        Prop_ObterNomeCliente
                        
                        Retorna o nome do cliente da proprietária.
                    
                    
                        Prop_ObterNomeModulo
                        
                        Retorna o nome de um módulo pelo seu índice.
                    
                    
                        Prop_ObterQtdInstanciasModWeb
                        
                        Quantidade de instâncias WEB para um determinado módulo.
                    
                    
                        Prop_ObterQtdLicencasModulo
                        
                        Quantidade de licenças de cada módulo (colaboradores para o Gestão de Pessoas, conexões para o Gestão Empresarial | ERP e controladores para o Acesso).
                    
                    
                        Prop_ObterQtdModulos
                        
                        Retorna a quantidade de módulos adquiridos pelo cliente.
                    
                    
                        Prop_ObterQtdUsuariosSistema
                        
                        Quantidade de usuários do sistema.
