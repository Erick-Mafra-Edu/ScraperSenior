            Criando Grades no Formulário
            Para adicionar uma grade, é necessário clicar sobre o formulário no painel à direita no SGI. Na tela de configuração que foi exibida, utilize o botão Inserir. 
            Se o tamanho já estiver definido, é possível informar a quantidade de linhas a ser visualizada pelo campo Linhas Visíveis.
            Para acrescentar colunas na grade, utilize o botão Adicionar para exibir a tela para configuração da coluna. Os demais botões, Alterar, Remover,Mover p/ cima e Mover p/ baixo, serão usados se houver a necessidade de ajustar na tela os campos que são adicionados.
            Selecionar a primeira coluna ao finalizar processo
                Determina se o foco deve ser posicionado na primeira coluna da Grade ao gravar ou cancelar alterações,  ao excluir um registro e quando se navega pelos registros da tela utilizando o navegador.
            Selecionar o primeiro registro ao atualizar dados 
                Determina se a grade será posicionada no primeiro registro quando seus dados forem atualizados.
            Usar a tecla ENTER para navegar pelos campos
                Determina se a tecla ENTER deve funcionar da mesma maneira que a tecla TAB para percorrer os campos da Grade.
            Respeitar condições de editabilidade 
                Determina se a opção de editabilidade dos campos da Grade deve ser respeitada, caso não esteja assinalada, os campos da Grade estarão editáveis, ainda que tenha uma Condição de editabilidade definida para o campo na Interface.

            Ao acrescentar uma coluna na grade (através do botão Adicionar), é preciso configurar suas propriedades na tela seguinte:
            
                O Campo correspondente determina o campo físico utilizado pela coluna; 
                O campo Máscara determina o tamanho do campo exibido na tela (quando este campo for deixado em branco assume a máscara do campo físico).
                O campo Maneira de ajudar o usuário a preencher deve ser escolhido entre as opções disponíveis.
            
            É possível adicionar uma caixa de seleção na grade. A máscara  CK['S':'N']: deve-se substituir os valores S e N pelos valores correspondentes à True e False. Este recurso só está disponível para versão web do sistema.
            Após confirmar esta operação, é preciso aplicar e testar para verificar o resultado.
            
                Importante
                
                É recomendável adicionar em Campos-chave todos os campos chaves da tabela base que não tenham sido herdados da tabela mestre, assim resultando em uma melhor performance na busca dos registros.
                Todas as grades criadas que referenciam tabelas, devem possuir uma ligação com a tabela principal da tela, de forma a ser consistida somente pela tabela mestre da tela. Demais ligações com outras tabelas podem ser utilizadas somente para auxiliar no posicionamento de registro ou filtros adicionais.
