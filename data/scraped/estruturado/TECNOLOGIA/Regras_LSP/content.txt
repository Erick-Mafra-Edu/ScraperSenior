            Regras LSP
            Esta tela permite localizar Regras LSP, através de informações contidas 
 no nome ou dentro da regra.
            Filtros da pesquisa: cada campo tem uma 
 máscara específica, caso não seja informado valor em algum campo, a pesquisa 
 será feita por todos, exceto para o campo Procurar 
 por, onde é necessário informar o conteúdo "[Todos]" 
 para que o campo seja desconsiderado. 
            Exemplo: 
                
                    Campo sistema, pode-se filtrar da seguinte forma: 
                    
                        FP (buscará pelo Administração de Pessoal); 
	 
                        F? (buscará pelos sistemas 
	 que começam com F); 
                        ?F (buscará pelos sistemas 
	 que terminam com F); 
                        F (apenas "F" 
	 sem o "?", é equivalente à "F?".  
                    
                    Este exemplo funciona para os campos Sistema, 
 Categoria e Extensão.
                
            
            
 Geral
            Nome
                Informe o nome 
 da regra LSP que será utilizado no executor, o nome deve ser único, e 
 auxiliar na identificação das demais regras registradas.
            Descrição
                A descrição deve conter a especificação 
 do que a regra faz, justificativas ou outros detalhes relevantes.
            Parâmetros
            Sistema
                 Informa-se a sigla do sistema pelo qual se deseja procurar, a máscara 
 para este campo é de dois caracteres.
            Categoria
                Campo disponível apenas para modelos 
 de relatório. Informa-se a categoria pelo qual se deseja procurar, a máscara 
 para este campo é de dois caracteres.
            Número
                 Informa-se os números dos modelos pelos 
 quais se deseja procurar, a máscara para relatórios são de três caracteres. 
 Para pesquisar por mais números de modelos coloque o valor "," 
 entre os números e se desejar pesquisar entre uma faixa de números separe-os 
 por "-". Exemplo: 1,2,15-25.
            Chaves
                É possível informar uma ou mais chaves 
 do EditCFG. Por padrão, Regras 
 e Rules já vem preenchidas e 
 é obrigatório informar ao menos uma chave.
            Procurar por
                Informa-se 
 o texto pelo qual se deseja pesquisar, ele procurará pelo texto dentro 
 dos modelos.
            Ao 
 informar algum conteúdo no campo Procurar por é preciso clicar no botão Adicionar, 
 assim o filtro de pesquisa será adicionado a lista logo abaixo.
            Em 
 Excluir é possível excluir algum 
 termo que foi selecionado na lista da opção Procurar por. O termo deve estar selecionado e ao clicar com o botão Excluir ele será eliminado da lista.
            Diferenciar maiúsculas de minúsculas
                Essa opção de filtragem é utilizada apenas 
 se for feita pesquisa no campo Procurar por. 
 Ela permite que sejam diferenciados ou não os caracteres que estiverem 
 em maiúsculo ou minúsculo. Por exemplo: ao utilizar a palavra "Regra" 
 e, se essa opção estiver marcada, não serão apresentados os relatórios 
 que contiverem apenas a palavra "Regra" no seu conteúdo.
            Sentença 
 inteira
                Essa opção de filtragem é 
 utilizada apenas se for feita pesquisa no campo Procurar por. Ela faz com que as palavras sejam analisadas 
 por inteiro, não dividindo-as. Por exemplo: ao informar um termo como 
 "parâmetros de inicialização" e, se essa opção estiver desmarcada, busca 
 por cada palavra separadamente não importando a ordem em que aparecem. 
 Caso esteja marcada, buscará os relatórios que contém estas palavras exatamente 
 nesta ordem.
            Ignorar 
 quebra de linha
                Essa funcionalidade 
 permite que as quebras de linhas inseridas nos relatórios sejam ignoradas, 
 fazendo com que várias linhas quebradas sejam consideradas como uma sentença 
 única.
            Busca 
 em
                Por padrão a opção "Regras" está marcada, mas essa funcionalidade 
 permite que a pesquisa também seja feita em todo o relatório.
            Plano 
 de ajustes (tempo por ocorrência) e Excluir 
 linhas sem ocorrência.
