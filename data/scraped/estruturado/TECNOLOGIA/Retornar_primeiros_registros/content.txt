            Retornar primeiros registros
            Comandos para retornar os primeiros registros da execução do select, conforme a quantidade determinada na cláusula FETCH FIRST N ROWS (ONLY, WITH TIES), executados no CBDS, LSP ou no fonte:
            
                FETCH FIRST N ROWS ONLY: retorna somente os primeiras registros conforme a quantidade especificada no parâmetro N.
                FETCH FIRST N ROWS WITH TIES: retorna os primeiras registros conforme a quantidade especificada no parâmetro N, e os registros idênticos ao do último registo encontrado, conforme coluna definida no ORDER BY. Para WITH TIES deve ser informado a cláusula GROUP BY.
            
            Sintaxe
            Exemplos de sintaxe dos comandos para adicionar nos selects a cláusula FETCH FIRST N ROWS ONLY.
            Selecionar registros
                
                    Selecionar apenas o primeiro registro
                    Comando SQLSENIOR2
                        
                            
                                SELECT A.NUMEMP, A.TIPCOL, A.NUMCAD FROM A010FUN A WHERE A.NUMEMP = 1 AND A.TIPCOL = 1 FETCH FIRST 1 ROWS ONLY
                            
                        
                    
                    Comando gerado SQLSERVER
                        
                            
                                SELECT TOP 1 a.numemp 'NUMEMP', a.tipcol 'TIPCOL', a.numcad 'NUMCAD' FROM a010fun a WHERE a.numemp = 1 AND a.tipcol = 1
                            
                        
                    
                    Comando gerado ORACLE11
                        
                            
                                SELECT T1.NUMEMP, T1.TIPCOL, T1.NUMCAD FROM (SELECT A.NUMEMP "NUMEMP", A.TIPCOL "TIPCOL", A.NUMCAD "NUMCAD", ROW_NUMBER() OVER (ORDER BY NULL) rowlimit_$$rownumber FROM A010FUN A WHERE A.NUMEMP = 1 AND A.TIPCOL = 1) T1 WHERE T1.rowlimit$$_rownumber <= 1
                            
                        
                    
                    Comando gerado ORACLE12
                        
                            
                                SELECT A.NUMEMP "NUMEMP", A.TIPCOL "TIPCOL", A.NUMCAD "NUMCAD" FROM A010FUN A WHERE A.NUMEMP = 1 AND A.TIPCOL = 1 FETCH FIRST 1 ROWS ONLY
