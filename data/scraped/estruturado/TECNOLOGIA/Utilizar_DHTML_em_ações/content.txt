                Como utilizar DHTML em ações
                
Para utilizar DHTML em ações, deve-se utilizar templates para gerar a página HTML. Nesses templates são  instalados os eventos que irão executar ações no servidor e devolver um script para manipulação dos elementos HTML no cliente.
                Procedimentos
                
                    Criar uma Interface chamada "testeDHTML";
                    Criar nesta Interface um campo chamado "Nome_Retorno" do tipo String 45;
                    Criar uma ação na Interface chamada "executaDHTML" com o código:
			 @ Campo que vem do HTML @	        Definir Alfa editXpto;	        ValStr = "<script>";>ValStr = ValStr + "parent.DADOS.document.getElementById('editXpto').onblur = '';";>@ A editXpto não existe na interface, ele etá definido apenas no template. @>ValStr = ValStr + "alert('Valor em editXpto é " + editXpto + "');";>@ Nome_Retorno é um campo definido na interface, e relacionado ao componente edNome do formulário. @	        @ Nesse caso não é necessário utilizar uma variável. Basta simplesmente acessar o campo da interface. @	        ValStr = ValStr + "alert('Valor em edNome é " + Nome_Retorno + "');";	        @ Exemplo DHTML @	        ValStr = ValStr + "parent.DADOS.document.getElementById('edNome').firstChild.style.color = 'blue';";	        ValStr = ValStr + "</script>";
		
                    Criar um formulário chamado "FrTesteDHTML";
                    Inserir uma linha de dados no formulário "FrTesteDHTML";
                    Inserir uma caixa de edição na linha de dados chamada "edNome" e relacionar com o campo da Interface Nome_Retorno;
                    Inserir os botões e respectivos rótulos btAprovar Aprovar e brReprovar Reprovar na linha de dados e relacionar com a ação "executaDHTML";
                    Utilizar no formulário FrTesteDHTML, o template:
			 <div class="templateBody">			   <table class="templateContent">			      <tr>			         <td class="templateLabelRow"> Nome: </td>			         <td class="templateContentRow"> [$edNome$] [$lbNomeRetorno$] </td>			      </tr>			      <tr>			         <td class="templateLabelRow"> EditXpto </td>			         <td class="templateContentRow"> <input name="editXpto" id="editXpto" type="text" /> </td>			      </tr>			      <tr>			         <td class="templateLabelRow"> Opções </td>			         <td class="templateContentRow"> [$btAprovar$] [$btReprovar$] </td>			      </tr>			   </table>			</div>			<script>			   document.getElementById("editXpto").onblur = function(){			      sendEvent("ActionEvent","executarDHTML");			   }			</script>
                
                
					
			O formulário ficará com a seguinte aparência:
		
                
                    
                        
                    
                
                 
            Após preenchidos os campos Nome e EditXpto, ao clicar nos botões, será executada a ação "executarDHTML" que irá exibir duas mensagens com os valores dos campos Nome e EditXpto. Também será alterada a cor da fonte do Nome para azul.
