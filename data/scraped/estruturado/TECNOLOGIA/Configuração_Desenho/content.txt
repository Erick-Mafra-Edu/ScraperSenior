                Configuração Desenho
                Esta tela permite configurar a forma e a cor de um controle desenho, tendo as seguintes opções:
                Desenho
                    Define a forma que o desenho vai assumir.
                Textura
                    Define o tipo de textura que o desenho vai assumir.
                Forma
                    Define a forma que a textura vai assumir.
                Estilo
                    Define o estilo que a linha vai assumir.
                Espessura
                    Define a espessura da linha externa (em pixel).
                Os botões desta tela, descritos abaixo, executam as seguintes operações:
                Cor Linha
                    Permite definir uma cor para a linha externa deste desenho.
                Cor Textura
                    Permite definir uma cor para a textura deste desenho.
