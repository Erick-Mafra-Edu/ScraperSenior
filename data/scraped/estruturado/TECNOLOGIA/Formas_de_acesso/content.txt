                Formas de Acesso
                Para optar por uma forma de acesso, é preciso considerar as necessidades e restrições que o ambiente apresenta. Na lista abaixo, há um comparativo das 4 opções de acesso disponibilizadas pela solução Senior:
                
                    BrowserAccess: Totalmente em HTML / JavaScript, executado por um navegador de páginas da internet. Todo o processamento é executado no servidor;
                    Cliente-Servidor: A instalação do sistema é feita completamente na estação, que se comunica diretamente com o banco de dados. Não utiliza o Middleware Senior;
                    WEB 5.0: É uma interface web que permite personalização dos HTMLs. Todo o processamento é executado no servidor;
                    WindowsAccess: É um aplicativo para desktop Windows. Todo o processamento é executado no servidor.
