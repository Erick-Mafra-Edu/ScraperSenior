            Configurações de Datasource ODBC
            Nas configurações do ODBC, são necessários alguns cuidados:
            
                
					Ao criar a fonte de dados no ODBC, crie como Fonte de Dados do Sistema. Se for criado em  Fontes de Dados de Usuário, o banco estará acessível apenas localmente;
                Durante a configuração do ODBC com banco de dados SQL Server será solicitado o Databasename utilizado, informe o database padrão do sistemas;
                Ao criar a configuração ODBC, marque a opção Executar conversão dos dados de caracteres;
                Ao chegar na tela de informação sobre a autenticação de login, selecione a opção para cadastrar utilizando o usuário do banco de dados. Informe o nome de usuário e senha para a conexão correta. É importante que, além do acesso ao banco de dados, o usuário tenha o acesso a base de dados também.
