            Tela de Seleção de Variáveis de Sistema
            As variáveis de sistema permitem que dados sejam extraídos e enviados ao sistema para aumentar a integração entre este e as regras de usuário. Para facilitar o uso destas variáveis, está disponível uma tela de seleção de variáveis de sistema que auxilia o usuário na tarefa de procurar e selecionar tais variáveis.
            Funcionamento da tela
            Na lista que aparece está a lista das variáveis de sistema que estão disponíveis para uso na regra.  As colunas com informações sobre as variáveis são:
            
                Nome: indica o nome da variável de sistema;
                Tipo: indica o número da variável. Nas regras, o tipo numérico são compatíveis com as variáveis do tipo número e os tipos cadeias de caracteres com o variáveis alfa;
                Ocorrências: este número indica se a variável é um vetor. Vetores são matrizes com apenas uma linha, onde cada coluna guarda um valor. Quando este número é zero (0) ou um (1), a variável é simples (não é um vetor). Quando este número é dois (2) ou mais, a variável define um vetor, que na regra deve ser acessado no formato var[n], onde var é o nome da variável e n é o número da ocorrência que deve ser acessada.
            
             Confirmando a seleção de uma variável
            Para confirmar uma seleção, é preciso clicar no botão s. Se uma variável estiver selecionada, o nome desta variável é inserido na regra na posição do cursor.
            Caso nenhuma variável esteja selecionada, será exibida uma mensagem alertando que não existe nenhuma informação selecionada.
            Cancelando a seleção
            Para cancelar a seleção de variável, basta clicar no botão Cancelar. Nenhuma informação é inserida na regra.
