            Permissões de telas para a Web 5.0
            Conceito
        
            Para exibir ou omitir itens de menu no Web 5.0 deve-se configurar a tela correspondente no ERP 
 Senior, já que para cada tela no WEB 5.0, existe uma tela equivalente no ERP. É importante salientar que o  WEB 
 5.0 ignora as permissões atribuídas ao item de grupo do menu ERP (tipo “Mercado ou Gestão de Vendas 
 etc...). A tabela de equivalência de telas pode ser encontrada abaixo:
            
                
                    Menu na Web 5.0.
                    Equivalência de permissões no Menu ERP Senior
                
                
                    
                        Vendas                   
                    
                
                
                    
                        Entrada de Pedidos
                    
                    
                        Por padrão o sistema utiliza como equivalência a tela Mercado > Gestão de Vendas > Pedidos > Agrupado (F120GPD), mas se o identificador de regra VEN-120PEDVE01; estiver ativo, será utilizada a tela Mercado > Gestão de Vendas > Pedidos > Individual (F120PED)
                    
                
                
                    
                        Pedidos via Grade*
                    
                    
                        Mercado > Gestão de Vendas > Pedidos > Grade de Produto (F120GRA)
